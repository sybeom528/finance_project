# HMM 레짐 분석 결과 보고서

> **분석 기간**: 2004-02-02 ~ 2026-02-27 (5,554 거래일)  
> **모델**: Gaussian HMM (covariance_type="full", n_iter=500)  
> **비교 대상**: 레짐 수 n=3, 4, 5  
> **최종 피처**: 검증 기반 정제 6개 피처 사용

---

## 1. 분석 개요

### 목적
HMM(Hidden Markov Model)을 사용해 금융 시장에 내재된 **숨겨진 상태(레짐)** 를 자동으로 탐지합니다.  
시장은 관측할 수 없는 상태(강세장, 약세장 등)에 따라 다른 통계적 특성을 보인다는 가정 아래,  
레짐 수를 3·4·5로 달리하며 최적 구성을 비교합니다.

### 분석 2단계 접근

| 단계 | 내용 |
|------|------|
| Phase 1 | 16개 피처로 탐색적 분석 → 4가지 검증(VIF, KW, HMM 분리력, Ablation) |
| Phase 2 | 검증 결과 기반 **6개 정제 피처**로 재분석 → 최종 권장 모델 |

### 데이터 구성

| 파일 | 내용 | 기간 |
|------|------|------|
| `ff_factors_daily.csv` | Fama-French 5팩터 + 모멘텀 (일별) | 2004~2026 |
| `macro_daily.csv` | VIX, WTI, 금리스프레드, 금·달러 (일별) | 2004~2026 |
| `daily_returns.pkl` | 개별 종목 일별 수익률 (5,554일 × 822종목) | 2004~2026 |
| `sector_etf.pkl` | 11개 섹터 ETF 가격 시계열 | 2004~2026 |

## 2. 분석 과정

### Step 1. 피처 엔지니어링 (Phase 1: 16개 → Phase 2: 6개)

원시 데이터에서 탐색적 분석을 위한 16개 피처를 구성 후, 검증을 통해 6개로 정제했습니다.

| 그룹 | 피처 | Phase 1 | Phase 2 | 이유 |
|------|------|---------|---------|------|
| 수익률·모멘텀 | `mkt_rf` | ✅ | ✅ **유지** | Ablation 핵심 기여 |
| 수익률·모멘텀 | `mkt_ret_ew` | ✅ | ❌ 제거 | mkt_rf와 VIF≈35~39 중복 |
| 수익률·모멘텀 | `ret_5d`, `ret_21d` | ✅ | ❌ 제거 | Ablation BIC 악화 |
| 변동성 | `vol_21d`, `vol_63d` | ✅ | ✅ **유지** | 4개 검증 모두 기여 |
| FF 팩터 | `smb`, `hml`, `mom` | ✅ | ❌ 제거 | Ablation BIC 악화 |
| 공포 지수 | `vix`, `vix_chg` | ✅ | ✅ **유지** | 분리력 최상위 |
| 금리 | `t10y2y` | ✅ | ✅ **유지** | KW 유의 + HMM 분리력 HIGH |
| 금리 | `t10y2y_chg` | ✅ | ❌ 제거 | Ablation BIC 최대 악화 |
| 원자재 | `wti_ret`, `gold_ret`, `dxy_ret` | ✅ | ❌ 제거 | Ablation BIC 악화 |

**최종 정제 피처 (6개)**: `mkt_rf`, `vol_21d`, `vol_63d`, `vix`, `vix_chg`, `t10y2y`

- NaN이 많은 초기 구간 제거 후 **5,554 거래일** 사용
- `StandardScaler`로 전 피처 표준화 (평균=0, 분산=1)

### Step 2. 모델 학습

```
GaussianHMM(covariance_type="full", n_iter=500, random_state=42)
```

- **covariance_type="full"**: 레짐마다 독립적인 6×6 공분산 행렬을 추정
  → 피처 간 상관관계까지 레짐별로 다르게 포착
- **EM 알고리즘** 500회 반복, n=3·4·5 모두 수렴(converged=True) 확인
- **레이블 부여**: 레짐을 평균 수익률 내림차순으로 정렬 후 이름 매핑

### Step 3. 최적 레짐 수 선택 (BIC)

BIC(Bayesian Information Criterion) = −2 × logL × T + n × ln(T)

BIC는 낮을수록 적합도와 모델 복잡도 간의 균형이 좋은 모델입니다.

#### Phase 2 (정제 6피처) BIC — 최종 결과

| 레짐 수 | logL | BIC | 수렴 |
|---------|------|-----|------|
| n=3 | −22,087.12 | 245,343,799 | ✅ |
| n=4 | −19,772.49 | 219,632,867 | ✅ |
| **n=5** | **−18,348.88** | **203,819,363** | **✅ ◀ 최적** |

→ BIC 기준 **n=5**가 최적이며, n=4→5 개선폭(약 1,580만)도 n=3→4 개선폭(약 2,570만)과 같은 방향으로 유의미합니다.

#### Phase 1 vs Phase 2 BIC 비교 (피처 정제 효과)

| 레짐 수 | Phase 1 (16피처) BIC | Phase 2 (6피처) BIC | 개선량 | 개선율 |
|---------|---------------------|---------------------|--------|--------|
| n=3 | 822,447,905 | **245,343,799** | −577,104,106 | **-70%** |
| n=4 | 786,832,518 | **219,632,867** | −567,199,651 | **-72%** |
| n=5 | 782,517,934 | **203,819,363** | −578,698,571 | **-74%** |

→ 피처 수를 16→6개로 줄였을 때 BIC가 **평균 72% 개선**. 불필요한 피처들이 모델에 상당한 노이즈를 유발하고 있었음이 입증됩니다.

## 3. 피처 적절성 검증 (Phase 1 탐색 분석)

> 처음부터 피처가 "적절하다"고 가정하지 않고, 4가지 방법으로 사후 검증했습니다.  
> 이 검증 결과가 Phase 2 피처 선택의 근거가 됩니다.

### 3-1. 다중공선성 진단 (VIF)

분산팽창계수(VIF) > 10이면 다른 피처들로 거의 예측 가능한 중복 정보입니다.

| 피처 | VIF | 진단 |
|------|-----|------|
| `mkt_ret_ew` | **39.3** | 🔴 심각 — mkt_rf와 거의 동일한 정보 |
| `mkt_rf` | **35.5** | 🔴 심각 — mkt_ret_ew와 중복 |
| `vix` | **24.8** | 🔴 심각 — vol_21d/63d와 연동 |
| `vol_63d` | **21.2** | 🔴 심각 — vol_21d와 높은 상관 |
| `vol_21d` | **20.6** | 🔴 심각 — vol_63d와 높은 상관 |
| `vix_chg` ~ `wti_ret` | 1.1~3.2 | ✅ 양호 |

**해석**: VIF 기준으로 상위 5개 피처에 심각한 다중공선성이 존재합니다.  
그러나 HMM은 회귀 모델과 달리 계수 추정이 아닌 **가우시안 분포 파라미터를 직접 추정**하므로,  
다중공선성이 계수 불안정성 문제를 일으키지는 않습니다.  
다만 공분산 행렬이 거의 특이(singular)에 가까워질 수 있어, `covariance_type="full"` 사용 시 주의가 필요합니다.

### 3-2. 레짐 간 분리력 검증 (Kruskal-Wallis 검정)

레짐 레이블(n=5 기준)을 그룹으로 삼아 각 피처의 분포 차이가 통계적으로 유의한지 검정했습니다.  
(Kruskal-Wallis는 정규성 가정 없는 비모수 검정 → 금융 데이터에 적합)

#### 유의한 피처 (p < 0.05) — 레짐을 실질적으로 구분함

| 피처 | H 통계량 | p값 | 해석 |
|------|---------|-----|------|
| `vol_63d` | **4,029** | < 0.001 | 레짐 간 변동성 차이가 가장 뚜렷 |
| `vix` | **3,921** | < 0.001 | 공포 지수로 레짐 구분에 핵심 역할 |
| `vol_21d` | **3,378** | < 0.001 | 단기 변동성도 강한 분리력 |
| `t10y2y` | **2,031** | < 0.001 | 금리스프레드가 레짐과 밀접 |
| `ret_21d` | 319 | < 0.001 | 21일 평균 수익률도 레짐별 차이 유의 |
| `ret_5d` | 43 | < 0.001 | |
| `vix_chg` | 25 | 0.0001 | |
| `smb` | 17 | 0.002 | |

#### 유의하지 않은 피처 (p ≥ 0.05) — 레짐 구분에 기여 미미

| 피처 | H 통계량 | p값 |
|------|---------|-----|
| `mkt_ret_ew` | 8.7 | 0.069 |
| `hml` | 8.2 | 0.084 |
| `wti_ret` | 7.1 | 0.128 |
| `gold_ret` | 6.1 | 0.192 |
| `mom` | 5.5 | 0.238 |
| `mkt_rf` | 5.1 | 0.280 |
| `dxy_ret` | 5.1 | 0.280 |
| `t10y2y_chg` | 4.7 | 0.322 |

**해석**: 16개 피처 중 **8개만 통계적으로 유의**합니다.  
`mkt_rf`, `mkt_ret_ew`, `hml`, `gold_ret` 등 일별 수익률 계열 피처들은 레짐 간 차이가 크지 않습니다. 이는 레짐이 수익률 수준보다는 **변동성·공포 수준**으로 구분된다는 것을 의미합니다.

## 5. 전이 행렬 해석

전이 행렬의 대각선(자기 지속 확률)은 레짐이 얼마나 지속되는지를 나타냅니다.

- **Bull → Bull**: 높은 자기 지속성 → 강세장은 한번 진입하면 오래 지속
- **Bear/Crisis → Bear/Crisis**: 역시 높은 지속성 → 위기도 단기에 끝나지 않음
- **Recovery → Bull**: 회복 구간에서 강세로 전환되는 자연스러운 흐름 확인
- 인접 레짐 간 전이 확률이 높음 → 레짐 변화가 점진적으로 발생
