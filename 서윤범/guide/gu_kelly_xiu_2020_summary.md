# Empirical Asset Pricing via Machine Learning
> Gu, Kelly, Xiu (2020) — *Review of Financial Studies* 33(5): 2223–2273

---

## 1. 연구 목적

| 목적 | 내용 |
|---|---|
| 핵심 질문 | 머신러닝으로 개별 주식의 **위험 프리미엄(기대 수익률)**을 얼마나 잘 측정할 수 있는가? |
| 방법론 과제 | 고차원 데이터(920개 변수) + 비선형 관계 동시 해결 |
| 경제적 목표 | R²_OOS를 넘어, **실제 포트폴리오 수익**으로 가치 증명 |

## 3. Feature Engineering

> **"We cross-sectionally rank all stock characteristics period-by-period and map these ranks into the [-1, 1] interval"**

- 매 시점마다 모든 특성 변수를 **횡단면(Cross-sectional) 랭크** → **[-1, 1]** 구간으로 정규화
- 목적: 레짐 전환(저변동성↔고변동성) 시 절대값 기반 패턴의 역전 방지

## 4. 비교 모델

| 유형 | 모델 |
|---|---|
| 선형 | OLS (920변수), OLS-3 (size·B/M·momentum만), Elastic Net, PCR, PLS |
| 비선형 | GLM (spline, 상호작용 없음), Random Forest, GBDT |
| 딥러닝 | MLP (은닉층 1~5개, NN1~NN5) |

## 5. 검증 방식

```
훈련(최초 18년) → 검증(이후 12년) → 테스트 OOS(30년)
```

- 훈련 샘플은 **1년씩 누적 확장(expanding window)**
- 검증 샘플은 **1년씩 뒤로 이동(roll-forward)**
- 시간 역순 학습 및 교차검증(CV) 금지

---

## 6. 평가 방법

### 손실함수
- **Huber Loss** — 금융 수익률의 heavy tail에 강건

### 표본 외 결정계수 (R²_OOS)

$$R^2_{OOS} = 1 - \frac{\sum (r_{i,t+1} - \hat{r}_{i,t+1})^2}{\sum r_{i,t+1}^2}$$

> 분모를 **역사적 평균이 아닌 0**으로 고정 → 나이브 예측(항상 0) 대비 성능 측정

### 포트폴리오 검증
- 가치가중 **롱-숏 십분위 스프레드** (상위 10% 매수 / 하위 10% 매도)

## 7. 핵심 결과

### R²_OOS (월별, OOS 기준)

| 모델 | 전체 | 대형주 Top1000 |
|---|---|---|
| OLS (920개) | −3.46% | −11.28% |
| OLS-3 | 0.16% | 0.31% |
| Elastic Net | 0.11% | 0.25% |
| Random Forest | 0.33% | 0.63% |
| GBDT | 0.34% | 0.52% |
| **NN3 (최고)** | **0.40%** | **0.70%** |

> R²가 작아 보이지만, 금융 수익률 예측에서 **0.3~0.4%도 경제적으로 매우 유의미**

## 8. 핵심 교훈

| 교훈 | 내용 |
|---|---|
| **비선형 + 상호작용** | 신경망·트리가 선형 모델 압도 — "비선형 자체"보다 **변수 간 상호작용 포착**이 핵심 |
| **정규화 필수** | 변수가 많으면 OLS 즉시 붕괴 — Elastic Net·PCR만 해도 회복 |
| **얕은 학습** | NN3 최고 성능, NN4·NN5는 오히려 하락 — 금융 데이터는 S/N 비율이 매우 낮음 |
| **시간 순서 엄수** | Cross-validation 금지, 시간 역순 학습 금지 |

---

## 9. 기여점 및 한계

**기여점**
- 대규모 실증으로 머신러닝의 **예측력 + 경제적 가치** 동시 증명
- 롱-숏 스프레드 전략만으로 연간 Sharpe 1.35 달성

**한계**
- 수익률 예측은 우수하나 **경제학적 인과관계 설명 불가**
- 샘플이 미국 주식 시장에 한정

## 10. 우리 프로젝트와의 괴리

| 논문 설계 | 우리 현실 | 문제점 |
|---|---|---|
| 주식 30,000종목 (동질 자산) | ETF 22개 (주식+채권+금 혼재) | Cross-section 비교 자체가 무의미 |
| 월별 패널 → 레이블 중복 없음 | 일별 패널 + 21일 타겟 → 21배 중복 | IS 독립 관측치 실제 1/21 |
| 18년 훈련 (수만 종목) | 2년 훈련 + 22 ETF = 528행 | 모델 학습 데이터 절대 부족 |

> **결론**: ETF 22개에 논문 방식을 직접 적용하는 건 구조적으로 맞지 않음.  
> 개별 주식 110개 + 월별 패널로 전환하는 것이 논문 방식에 훨씬 근접.
