## 5. 공분산 추정 방법

### 확정 구조
```
Level 1 (자산군 간): 블록 대각 — 대표 ETF 3개(SPY, AGG, GLD) 간 3×3
Level 2 (주식 그룹): PCA 팩터 모형(K 자동 선택) + 레짐 조건부
Level 2 (채권 그룹): Ledoit-Wolf (N=4, 안정)
Level 2 (대안 그룹): 직접 추정 (N=2, 파라미터 1개)
```

### PCA 팩터 모형 상세
- K 자동 선택: 누적 설명 분산 >= 80%인 최소 K (하한 3, 상한 10)
- IS 윈도우별 K 변동 허용 (시장 환경 적응)
- 각 IS에서 선택된 K 값을 로그로 저장

### 레짐 조건부 Ledoit-Wolf
- IS 내 HMM 레짐별로 Σ_stable, Σ_crisis 분리 추정
- 경보 L0~L1 → Σ_stable, L2~L3 → Σ_crisis
- Fallback (위기기 관측일 부족 시):
  - >= 48일: 정상 분리 추정
  - 20~47일: Σ_crisis = Σ_stable × 1.5 (스트레스 스케일링)
  - < 20일: 단일 공분산 (레짐 분리 포기)

### 채택/탈락 근거 요약
- Ledoit-Wolf: Level 2 채권/대안에서 채택 (N 작음, 안정적)
- 블록 대각: Level 1에서 채택 (이중 계산 차단, 파라미터 3개)
- PCA 팩터 모형: Level 2 주식에서 채택 (N=24 → LW만으론 비율 1.8:1 불안정)
- DCC-GARCH: 미채택 (N=24에서 파라미터 700+, 수렴 불안정, 계산 수 시간)
- 랜덤 행렬 이론: 미채택 (구현 복잡, 금융 검증 사례 제한)
- Graphical LASSO: 미채택 (λ 튜닝 필요, 역공분산 기반으로 변환 필요)
- 팩터 모형이 DCC-GARCH + 랜덤 행렬의 핵심 효과를 포함하면서 구현 간단

## 7. 개별 종목 선정 및 재구성 규칙

### 선정 규칙
```
1. 후보군: 매년 1월 1일 기준 미국 대형주 시총 상위 100개
2. 필터: 거래대금 $500M+, 상장 2년+, 구조 변화 없음, 가격 $5+
3. 선정: 8개 GICS 섹터 각 시총 1위
4. 완충: 기존 보유 종목이 섹터 2위 이내면 교체하지 않음
5. 교체 시: 왕복 거래비용 30bps 부과
```

### "섹터별 시총 1위" 근거
- 기계적: 사람의 주관 배제
- 중립적: 성장주/가치주 편향 없음 (성향 차이는 최적화에서 표현)
- 섹터 분산 자동 보장
- 공격형의 성장주 선호: 높은 주식 비중 + 높은 개별 상한으로 충분히 표현

### 생존자 편향 방지
- NVDA(2015년 시총 200위) 같은 사후 성공 종목이 자동 탈락
- "매년 그 시점의 1위"를 사용하므로 미래 정보 개입 없음

### 교체 완충(Buffer) 규칙 근거
- 섹터 1위-2위 상관관계 평균 0.75~0.85 (거의 동일 노출)
- 교체 비용(왕복 30bps) > 기대 초과 수익(연 0.1~0.3%) → 교체하지 않는 것이 합리적
- 3위 이하로 밀림 = 구조적 변화(사업 악화, 경쟁사 부상) → 교체 합리적
- 기대 교체 빈도: 연 1~2개 종목

### 재구성 규칙 타당성 검증 (Perturbation Test)
- 원래 규칙 + 변형 5가지(시총 2위, 랜덤 1~2위, 시점 변경, 완충 없음, 완충 3위)
- 통과 기준: |Sharpe_원래 - Sharpe_변형| / Sharpe_원래 < 10%
- 10% 근거: 거래비용 영향 범위(Sharpe 0.03~0.08)와 동일 수준
- 학술적 기준은 추후 확인 시 변동 가능

## 9. 투자자 성향 프로파일

### generate_profile(gamma) 연속 함수
```python
def generate_profile(gamma):
    t = (gamma - 1) / 9  # 0(공격) ~ 1(보수)
    return {
        'gamma': gamma,
        'target_vol':  0.22 - 0.15 * t,
        'max_equity':  0.90 - 0.60 * t,
        'min_bond':    0.00 + 0.40 * t,
        'max_mdd':    -(0.35 - 0.25 * t),
        'max_single':  동적 산출 (아래 10장 참조),
    }
```

### 4개 프리셋
| 성향 | γ | 목표 변동성 | 주식 상한 | 채권 하한 | MDD 허용 |
|------|---|-----------|----------|----------|---------|
| 보수형 | 8 | 7% | 30% | 40% | -10% |
| 중립형 | 4 | 10% | 55% | 20% | -18% |
| 적극형 | 2 | 15% | 75% | 10% | -25% |
| 공격형 | 1 | 22% | 90% | 0% | -35% |

### gamma 연속 곡선 분석
- gamma 1.0~10.0 (0.5 단위, 19개 지점)
- 각 지점에서 MV 최적화 + W-F 백테스트 → Sharpe/MDD 곡선
- Config별 최적 gamma 비교 → "대안데이터가 최적 리스크 수준을 낮추는가?"

### 설문 기반 리스크 스코어: 이번 프로젝트 범위 밖
- 이유: 프로젝트 목적(대안데이터 검증)과 불일치, 행동 데이터 없이 검증 불가, 분석 결과에 영향 없음
- generate_profile(gamma)가 있으므로 향후 설문→gamma 매핑 함수만 앞단에 추가하면 확장 가능

## 10. 비중 상한 (HHI 기반)

### 산출 공식

```
비중 상한 = min(MDD 역산값, HHI Cap)

MDD 역산: (MDD 허용 × 1/3) / 극단 하락률 × 배수
HHI Cap:  HHI 기준을 충족하는 최대 개별 비중
```

### 자산 유형별 배수 (역사적 데이터 기반)

극단 하락률 수집 (2008 GFC, 2020 COVID, 2022 금리인상):
- 인덱스 ETF: 1개월 최악 -35% (SPY -33.9%, IWM -41.5% 등의 보수적 중앙값)
- 섹터 ETF: 1개월 최악 -45% (XLE -57.9%, XLF -38.4% 등의 중앙값)
- 개별 종목: 수개월 최악 -50% (META -76.7%, BA -72.4% 등의 보수적 중앙값)

배수 = 개별 종목 극단 하락(50%) / 해당 유형 극단 하락
- 인덱스 ETF: 50/35 = 1.43 → **1.4** (반올림)
- 섹터 ETF: 50/45 = 1.11 → **1.1** (반올림)
- 개별 종목: 50/50 = 1.00 → **1.0** (기준)

### HHI(Herfindahl-Hirschman Index) 기반 Cap

HHI = Σ(w_i²), DOJ 시장 집중도 기준 차용
- 보수형/중립형: HHI < 0.10 (비집중) → 최대 비중 ~15%
- 적극형: HHI < 0.15 (보통 집중) → 최대 비중 ~20%
- 공격형: HHI < 0.20 (고집중 미만) → 최대 비중 ~25%

### 최종 확정 비중 상한표

| 성향 | 개별 종목 | 섹터 ETF | 인덱스 ETF | 바인딩 근거 |
|------|----------|---------|-----------|-----------|
| 보수형(γ=8) | 6.7% | 7.3% | 9.3% | MDD 역산이 HHI보다 엄격 |
| 중립형(γ=4) | 12.0% | 13.2% | 15.0% | 인덱스만 HHI cap |
| 적극형(γ=2) | 16.7% | 18.3% | 20.0% | 인덱스만 HHI cap |

## 11. 경보 시스템

### VIX 우선 구조의 근거
1. 반응 속도: VIX(옵션) = 수초, HY스프레드 = 수시간, ICSA = 1주, Sahm = 1개월
2. 보편성: 거의 모든 유형의 위기에 VIX 반응 (다른 지표는 특정 채널만)
3. Granger 검정: VIX 계열(2,3,6위)이 상위권 장악

### VIX만으로 부족한 상황 (비VIX 독립 트리거 필요성)
- 느린 위기: 2022 금리인상 VIX 30 내외인데 SPY -25%
- 신용 경색: 2023 SVB 초기 HY가 VIX보다 1~2일 선행
- 경기 침체 초기: 매크로 지표 악화 중 VIX 아직 저위

### 4가지 Config

| Config | 복합 스코어 | 독립 트리거 |
|--------|-----------|-----------|
| A | 없음 | VIX >= 20/28/35 (고정) |
| B | 없음 | A + VIX_contango < 0 |
| C | 백분위 기반 (IS 내 P75/P90/P97) | + HY 5일변화>+0.5%p, 곡선 역전 전환, Sahm>=0.50, 깊은 백워데이션<-0.05 |
| D | C + 디바운싱 (5일 중 3일) | C의 독립 트리거는 디바운싱 비대상 (구조적 이벤트는 즉시 반영) |

- VIX_level 25%, VIX_contango 15%, HY_spread 20%, yield_curve 10%, SKEW_level 10%, claims_zscore 10%, Cu_Au_ratio_chg 10%
- 임계값: IS 윈도우 내 백분위 기반 (v2의 고정 25/50/75에서 변경)

### 독립 트리거 근거
| 트리거 | 임계값 | 역사적 근거 |
|--------|--------|-----------|
| HY 5일 변화 > +0.5%p | 0.5%p | 2020 COVID +1.2%p/5일, 2023 SVB +0.6%p/5일 |

근거: MDD 여유 기반 역산. 각 경보 등급의 기대 추가 하락(L1:-10%, L2:-20%, L3:-30%)에서 MDD 한도의 1/3 이내로 개별 종목 손실 제한하는 비중 축소율 역산.

### 경보 → 비중 전환의 이중 경로

## 13. 최적화 전략

### 역할
- MV(Mean-Variance): 주 전략 (γ가 직접 반영, 성향 맞춤)
- RP(Risk Parity): 비교 전략 (μ 추정 불필요, MV 과적합 검증용)
- HRP: 비교 전략 (공분산 역행렬 불필요, 가장 강건)

### Ablation에서의 적용
- 메인: MV로 4 Config × 3 재구성빈도 × 4 성향
- 보조: RP/HRP로 중립형(γ=4)만 → MV 대비 강건성 확인

## 14. Ablation Study

### 비교 축
- 축 1 경보 방식: A(VIX-only), B(+Contango), C(복합+독립트리거), D(C+디바운싱)
- 축 2 재구성 빈도: 고정, 연간, 분기
- 축 3 성향: 보수(γ=8), 중립(γ=4), 적극(γ=2), 공격(γ=1)

### 핵심 비교 (12개 백테스트)
- 중립형 기준, 재구성 효과: Config A × {고정,연간,분기} + Config B × {고정,연간,분기} = 6개
- 연간 기준, 성향 효과: 4성향 × Config B = 4개
- 벤치마크 2개: Baseline EW, SPY 100%

### 통계 검정
- 주 검정: Bootstrap Sharpe 차이 95% CI (5,000회)
- 보조: Bonferroni 보정 (3회 비교 → α'=0.0167) 또는 FDR
- 효과 크기: Cohen's d (< 0.2 미미, 0.2~0.5 소, 0.5~0.8 중, >= 0.8 대)

### 스트레스 테스트 시나리오 (11개)
