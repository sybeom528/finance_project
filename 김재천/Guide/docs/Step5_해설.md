## 2. 사전 지식

### 📚 용어 사전

| 용어 | 쉬운 설명 |
|------|---------|
| **VaR (Value at Risk)** | "N일 동안 X% 확률로 넘어서지 않을 손실" |
| **CVaR** | "VaR를 넘어선 경우의 평균 손실" (Expected Shortfall) |
| **역사적 VaR** | 과거 수익률 분포에서 직접 계산 |
| **모수적 VaR** | 정규분포 가정 후 계산 |
| **스트레스 테스트** | 가상/역사적 극단 시나리오에서 포트폴리오 성과 평가 |
| **MDD** | Maximum Drawdown (최대 낙폭) |
| **Tail Risk** | 꼬리 리스크, 극단 이벤트 위험 |

## 3. VaR / CVaR 개념

### 🎓 VaR의 직관적 이해

**일상 비유 - 기상 예보**:
- "내일 비올 확률 95%" → 95% 확률로 비온다
- "VaR 95% = -2%" → **95% 확률로 손실이 2% 이하**

**수학적 정의**:
```
VaR_α = -inf{x : P(Loss ≤ x) ≥ α}

예: VaR 95% 일별 = -1.1%
→ 일별 손실이 -1.1%를 넘을 확률 5%
```

### 🎓 CVaR의 의미

**VaR의 한계**:
- "95% 커트라인" 넘으면 얼마나 잃을지 모름
- -2%일 수도, -30%일 수도

**CVaR 해결**:
```
CVaR_α = E[Loss | Loss > VaR_α]

예: CVaR 99% = -2.8%
→ 최악의 1% 구간에서 평균 -2.8% 손실
```

### 📊 두 지표 비교

| 지표 | 장점 | 단점 |
|------|------|------|
| VaR | 직관적 (쉬운 해석) | 꼬리 정보 없음 |
| CVaR | 꼬리 리스크 반영 | 추정 어려움 (극단 이벤트 드물어) |

### 🎯 역사적 VaR vs 모수적 VaR

**역사적 VaR (Non-parametric)**:
- 과거 데이터 **분포 그대로 사용**
- 실제 꼬리 패턴 반영
- 단점: 과거에 없던 이벤트 예측 못함

**모수적 VaR (Parametric)**:
- 수익률이 **정규분포** 따른다 가정
- VaR = μ - 1.645σ (95% 수준)
- 단점: 재무 수익률은 정규분포 아님 (fat-tail)

**우리 선택**: **역사적 VaR** (실제 데이터 존중)

## 8. FAQ

### ❓ Q1. VaR 95%와 99% 중 어떤 게 중요한가요?

**A**: 둘 다 봐야 함:
- VaR 95%: 일상적 리스크 관리용
- VaR 99%: 극단 이벤트 대비용 (바젤 규제 기준)

### ❓ Q2. 역사적 VaR의 한계는?

**A**: 과거에 없던 이벤트 예측 불가.
- 예: 2020 COVID는 2019년까지 역사적 VaR로는 예측 못했음
- → 스트레스 테스트 (가상 시나리오)로 보완

### ❓ Q3. MDD가 VaR보다 큰 이유?

**A**: MDD는 **누적** 손실, VaR는 **일별**:
- VaR 99% (일) × 63일 = 최대 -X% (누적 시)
- 그러나 실제 MDD는 더 작음 (반등 있어 상쇄)

### ❓ Q4. 왜 가상 시나리오는 보수적 수치인가요?

**A**: 정적 배분 가정. **실제 투자자는 경보 보고 주식 축소** (경로 1) 가능. Step 7/9에서 동적 대응 효과 측정.

### ❓ Q5. 스트레스 테스트에서 공격형이 가장 나쁜 이유?

**A**: 주식 비중이 높기 때문. 공격형 (주식 90%까지)은 위기 시 타격 최대. 보수형 (주식 43%)은 채권·금이 상쇄.
