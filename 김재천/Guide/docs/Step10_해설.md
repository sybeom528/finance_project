## 2. 사전 지식

### 📚 통계 검정 용어 사전

| 용어 | 쉬운 설명 | 본 프로젝트 사용 |
|------|---------|---------------|
| **Bootstrap** | 데이터를 수천 번 재추출하여 분포 추정 | 5,000번 반복 |
| **신뢰구간 (CI)** | "95% 확률로 참값이 이 범위 안에 있다" | 95% CI |
| **p-value** | 우연히 이 정도 차이가 나올 확률 | α=0.05 임계 |
| **유의성 (Significance)** | 통계적으로 의미 있는 차이 | CI가 0 배제하면 유의 |
| **Bonferroni** | 다중 비교 시 기준을 엄격하게 (보수적) | α/K, K=비교 수 |
| **FDR** | 거짓 발견 비율 통제 (실용적) | Benjamini-Hochberg |
| **IR (Information Ratio)** | 초과수익/추적오차 (재무 실무 표준) | Grinold-Kahn 기준 |
| **ΔSR (Sharpe 차이)** | 두 전략의 연율 Sharpe 차이 | 실무 임계 0.2 |
| **Cohen's d** | 효과 크기 (심리학 표준, 재무에는 부적합) | v4.1에서 IR로 교체 |
| **Modular Decomposition** | 경로별 순수 기여도 분해 | M1, M2, M3 비교 |

## 4. 주요 개념 심화 설명

### 🎓 개념 1: Bootstrap 검정

**일상 비유 - 여론 조사**:
- 1,000명 원본 데이터에서 1,000명을 **복원 추출** (같은 사람 여러 번 뽑힐 수 있음)
- 이 과정을 5,000번 반복 → 5,000개 샘플링 결과
- 각 샘플에서 Sharpe 차이 계산 → 분포 얻음
- 분포의 2.5%, 97.5% 지점이 95% 신뢰구간

**왜 필요?**: 실제 분포 모양(정규분포 등)을 가정하지 않고 검정 가능

**우리 설정**:
```python
n_boot = 5000
# 일별 수익률 시계열을 동일 인덱스로 두 전략 동시 리샘플링 (paired)
# → 시장 조건 공유하므로 보다 공정한 비교
```

### 🎓 개념 2: 다중 비교 보정 (Bonferroni vs FDR)

**일상 비유 - 복권 추첨**:
- 1회 복권: 당첨 확률 낮음 (유의성 기준 엄격)
- 100회 복권: 1회라도 당첨될 확률 90%+ (1회 기준으로 보면 우연 당첨 많아짐)
- 복권 비유 → 통계 비유: **검정을 많이 하면 우연히 유의한 결과 증가**

**Bonferroni (보수적)**:
```
원래 α = 0.05
보정 α = 0.05 / K  (K=비교 수)
3개 비교면 α = 0.0167 (98.33% CI 사용)
```

**FDR (실용적)**:
```
"전체 중 몇 %까지 거짓 발견 허용할까?"를 통제
더 덜 보수적 → 검정력 보존
```

**우리 설정**: 두 방법 모두 적용하여 robustness 검증

### 🎓 개념 3: Cohen's d vs IR + ΔSR (v4.1 변경)

**왜 Cohen's d를 폐기했나?**

**일상 비유 - 측정 단위**:
- "방 면적" 측정: 3.3㎡ = 1평, 50㎡ 주택은 "보통"
- "주방 면적" 측정: 3㎡도 "크다" (기준이 다름)

**Cohen's d의 문제**:
- 원래 심리학·행동과학용 (cross-sectional 단일 측정)
- d = (평균 차이) / 표준편차
- **재무의 일별 수익률은 노이즈 비율이 높아 d가 거의 항상 0.01 이하**
- 모든 전략이 "미미(negligible)"로 분류되어 해석 불가

**대체안 1: Information Ratio (IR)**
```
IR = annualized(평균 초과수익) / annualized(초과수익 표준편차)
```
**Grinold-Kahn 기준**:
- IR < 0.3: 미미
- IR 0.3~0.5: 양호
- IR 0.5~1.0: 우수
- IR ≥ 1.0: 예외적

**대체안 2: ΔSR 실무 임계**
```
|ΔSR| < 0.1 → 미미
|ΔSR| 0.1~0.2 → 경계선
|ΔSR| 0.2~0.3 → 실질 개선
|ΔSR| 0.3~0.5 → 매우 유의
|ΔSR| ≥ 0.5 → 탁월
```

### 🎓 개념 4: 경로 기여도 분해 (Modular Decomposition)

**일상 비유 - 요리 실험**:
- 평범한 김치찌개 = **M0**
- 고기 추가 = **M1** (고기만)
- 참치 추가 = **M2** (참치만)
- 고기 + 참치 = **M3** (둘 다)

**효과 분해**:
```
고기의 기여 = M1 - M0 (얼마나 맛있어졌나?)
참치의 기여 = M2 - M0 (얼마나 맛있어졌나?)
둘 다의 기여 = M3 - M0
상호작용 = M3 - M1 - M2 + M0 (시너지 또는 상쇄?)
```

**v4.1 추가**: **M3 - M1** (참치가 고기 위에 추가 가치 있나?)

### 🎓 개념 5: Leave-One-Window-Out (LOO) 견고성 검증

**일상 비유 - 수업 평가**:
- 학생 A의 평균 성적: 90점
- 만약 1번 시험을 제외하면?: 평균 95점
- 만약 5번 시험을 제외하면?: 평균 88점
- → **특정 시험이 평균을 크게 좌우**하면 "취약"

**우리 적용**:
- 31개 윈도우 중 하나씩 제외하고 Sharpe 재계산
- 최대 |ΔSharpe| < 0.05 → **견고함**
- 최대 |ΔSharpe| > 0.10 → **특정 기간 의존**

### 🤔 주요 결정 사항

#### 결정 1: Cohen's d → IR + ΔSR 교체 (v4.1)

**피드백 원문**:
> "Cohen's d는 일별 로그 수익률의 낮은 SNR로 인해 모두 '미미'로 분류되어 해석 무용"

**효과**: 차별화된 해석 가능 (M2: 미미/음, M1: 양호 등)

#### 결정 2: M3 vs M1 직접 검정 추가 (v4.1)

**피드백 원문**:
> "M0만 기준으로 비교하면 M3의 실질 부가가치(경로 2 추가 효과) 측정 불가"

**채택**: 64 비교로 확장 (M1/M2/M3 vs M0 + M3 vs M1)
**결과**: **M3 - M1 < 0** 확인 → 경로 2 역효과 실증

#### 결정 3: 가상 시나리오 컬럼명 수정

**피드백 원문**:
> "'M3 예상 1일 수익률'은 오해 소지, 실제로는 정적 배분 근사"

**채택**: 컬럼명을 "정적배분 1일 shock (근사)"로 변경 + 명시적 주석

#### 결정 4: 매매비용 완화 판정 논거 정교화

- **"구조적 효과"가 아닌 "시장 국면별 경로 1의 차등 기여"**

## 9. FAQ

### ❓ Q1. FDR 유의율 18.8%면 M1을 신뢰할 수 있나요?

**A**: 주의 필요:
- 유의 3개/16 비교 — 대부분 "통계적으로 확신 못함"
- 다만 **ΔSharpe 부호는 전부 양수** (16/16)
- **효과 방향은 일관**되나 **크기가 불확실**
- 실전 운용 시 소액 테스트 후 확대 권장

### ❓ Q2. 왜 Bonferroni를 쓰나요?

**A**: **거짓 발견 방지**:
- 48개 비교 중 우연히 유의한 결과 기대: 48 × 0.05 = 2.4개
- Bonferroni로 보정해야 "진짜 효과"만 걸러냄
- 보수적이어서 검정력 희생 → FDR도 병행

### ❓ Q3. 경로 2가 왜 실패한 건가요?

**A**: [Step 9 해설 Section 8 참조]:
1. Σ 차이가 MV 비중에 반영 안 됨
2. 재최적화 비용 > 이론적 이득
3. 경로 1과 정보 중복

### ❓ Q4. v4.0 초안의 "Sharpe 1.118"은 어떻게 된 건가요?

**A**: 다음 수정 누적 결과:
- 경로 2 월 단위 재전환 → M3 Sharpe 하락 (1.118 → 1.027)
- 60/40 벤치마크 비용 정교화 → 상대 Sharpe 기준 재조정
- **최종 M1_보수형_ALERT_B (1.064)가 Multi-criteria 1위**로 부상

### ❓ Q5. IR 0.42는 어떤 수준인가요?

**A**: Grinold-Kahn 기준:
- < 0.3: 미미
- **0.3~0.5: 양호 ⭐** (M1이 여기)
- 0.5~1.0: 우수
- ≥ 1.0: 예외적

재무 펀드 매니저 상위 20% 수준으로 해석 가능.

### ❓ Q6. 최소 유지 기간 3일 적용 시 어떻게 되나요?

**A**: v4.2 후속 과제. 이론적 예상:
- 경보 변동 연 28회 → 약 15회로 감소
- 거래비용 연 0.83% → 0.44% 절감
- M1 Sharpe 예상: 1.064 → 1.08~1.10 (상승)
- 단, 위기 초기 대응 3일 지연 리스크 존재

### ❓ Q7. 다른 자산 조합으로도 같은 결과일까요?

**A**: 불확실. 이 결과는 특정 30자산 + 2018-2025 기간에 특이할 수 있음:
- 글로벌 분산, 신흥국 포함 시 다른 결과 가능
- 2008 GFC 등 과거 위기 포함 시 경로 2 효과 다를 수 있음
