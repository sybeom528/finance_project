## 1. 배경과 목적

### 🎯 숫자만으로는 부족한 이유

Step 10에서 "최우수 전략 Top 10"을 확정했지만 독자 입장에서 궁금한 것:

| 질문 | 숫자 답변 | 진짜 궁금한 답 |
|------|---------|------------|
| 얼마나 벌었나? | "Sharpe 1.064" | **언제** 잘 벌었나? |
| 얼마나 잃었나? | "MDD -15.53%" | **왜** 그때 손실이 멈췄나? |
| 어떻게 운영하나? | "경보→축소" | **어떤 자산**을 줄였나? |

**일상 비유 — 다이어트**:
- "10kg 감량 성공"은 숫자
- 실제로는 "언제 무엇을 바꿨는가?" (운동 종류, 식단 변경 시점, 스트레스 요인)
- 이 "**시간 궤적과 원인**"을 봐야 재현 가능

### 🎯 Step 11의 목표

> **"Top 10 전략이 2018~2025 동안 실제로 언제·무엇을·왜 바꿨는지 시각적으로 재구성"**

## 2. 사전 지식

### 📚 용어 사전

| 용어 | 쉬운 설명 | 비유 |
|------|---------|------|
| **자산 비중 (Weight)** | 전체 자본에서 각 자산이 차지하는 % | 반찬 가짓수별 젓가락질 비율 |
| **Stacked Area Chart** | 비중을 시간축으로 쌓아 그리는 차트 | "포트폴리오 지층 단면도" |
| **Turnover** | 리밸런싱 시 바뀌는 비중 총합 | 옷장 정리 때 교체한 옷의 비율 |
| **앵커(Anchor) 자산** | 꾸준히 높은 비중으로 유지하는 자산 | 집에 항상 있는 쌀 |
| **반응(Reactive) 자산** | 시장 상황에 따라 크게 변하는 자산 | 계절별로 바뀌는 옷 |
| **WF 경계** | 분기 리밸런싱 경계선 (3개월마다) | 계절 바뀌는 날 |
| **Σ 전환** | 월 단위로 공분산 선택 변경 (M3만) | 월별 체크업 |
| **경보 레벨** | 0(정상) / 1(주의) / 2(경계) / 3(위기) | 신호등 + 비상 모드 |

## 4. 시각화 8종 상세 설명

### 🖼️ 시각화 1: 통합 대시보드

**파일**: `step11_01_dashboard.png` (2×3 패널)
**제목**: "최우수 전략 M1_보수형_ALERT_B의 시간 여행"

**읽는 법**:
- **(1) 누적 자본**: 빨간 선 = M1, 회색 점선 = EW(벤치마크). M1이 더 높으면 경보 전략의 가치
- **(2) 자산군 비중**: 주식(빨강)·채권(파랑)·금(주황). 시간 따라 어떻게 섞이는지
- **(3) 경보 레벨**: 초록(정상) / 노랑(주의) / 주황(경계) / 빨강(위기). 경보 이벤트 타임라인
- **(4) HMM 레짐**: 시장 상태 자동 분류
- **(5) 주식 총비중**: 경보 발동 시 떨어지는 패턴
- **(6) Drawdown**: 손실 시계열 + 경보 상승 시점(점선)

**핵심 관찰**: 패널 3·5·6을 **세로로 비교**하면 "경보 상승 → 주식 감소 → 손실 제한" 인과사슬이 한눈에 드러납니다.

**비유**: "운전 기록 블랙박스" — 사고(손실)가 났을 때 어떤 신호(경보)가 먼저 울렸고, 운전자가 어떻게 반응(비중 축소)했는지 한 장에서 재구성.

### 🖼️ 시각화 8: 앵커 vs 반응 자산 산점도

**파일**: `step11_08_asset_stability.png`
**제목**: "M1_보수형_ALERT_B: 앵커 자산 vs 반응 자산 해부도"

**읽는 법**:
- **X축**: 평균 비중 (오른쪽 = 큰 비중)
- **Y축**: 비중 변동성 (위 = 많이 변함)
- **색**: 자산군 (빨강=주식, 파랑=채권, 주황=금)
- **버블 크기**: 총 기여도 (평균 비중 × 누적 수익률)

**4사분면 해석**:
- **오른쪽 아래** = **앵커** (큰 비중 + 안정)
- **왼쪽 위** = **반응 버퍼** (작은 비중 + 변동 큼)
- **오른쪽 위** = 주력 + 변동 (주식 중에서는 가장 적극 운용)

**핵심 관찰**:
- 채권(AGG, TLT) = 오른쪽 아래 (앵커)
- 금(GLD) = 중간 우측 (부앵커)
- 개별 주식 → 왼쪽 위 (반응 버퍼)

**비유**: "오케스트라 악기별 역할 분석" — 베이스(앵커, 항상 밑 깔기) vs 솔로 악기(반응, 상황별 등장).

## 7. 결과 해석

### 💡 핵심 발견 5가지

#### 1. **경보 시스템의 실전 증명**
- 시각화 5에서 경보 레벨과 주식 비중이 체계적으로 역상관
- L0(정상) 주식 40% → L3(위기) 주식 16%
- "경로 1이 실제로 작동한다" 시각적 증거

#### 2. **앵커-반응 이원 구조**
- 채권 4개 (TLT/AGG/SHY/TIP) + 금 (GLD) = 안정 앵커
- 주식 24개 = 경보 반응 버퍼
- → 보수형 설계의 실증적 해부

#### 3. **경로 2(M3)의 시각적 무효성 재확인**
- 시각화 6: M1과 M3의 비중 흐름이 거의 동일
- Σ 전환 경계에 미세한 점프만 존재
- → Step 10 통계 결론과 정합

#### 4. **Turnover 주 원인 = 경보**
- 시각화 7: 주황(경보 변동)이 총 turnover의 **대부분**
- 분기 리밸런싱(파랑)은 소량
- Σ 전환(보라)은 M3에서만 존재, 소량

#### 5. **자산 기여도 불균형**
- 시각화 8에서 대형 버블(기여도 ↑) = 주로 채권·금
- 주식 개별주는 작은 버블 = 기여 적음
- → 보수형 포트폴리오의 실제 동력은 **안전자산**

### 📊 앵커 자산 Top 5 (M1_보수형_ALERT_B 기준 예시)

| 자산 | 평균 비중 | 변동성 | 역할 |
|------|---------|------|------|
| AGG | 20~25% | 낮음 | 메인 앵커 |
| TLT | 15~20% | 중간 | 장기채 완충 |
| GLD | 10~15% | 중간 | 금 안전자산 |
| SHY | 10~15% | 낮음 | 단기채 현금 대안 |
| TIP | 8~12% | 낮음 | 인플레 방어 |

## 8. FAQ

### ❓ Q1. 왜 Top 10 재시뮬이 약 3분 걸리나요?

**A**: 월 단위 Σ 전환(M3)이 분기당 최대 3회 MV 최적화. 31 윈도우 × 10 전략 × 가변 최적화 횟수 = 약 수백회 MV 최적화 누적.

### ❓ Q2. 시각화 2의 30색 구분이 어려운데 어떻게 읽어야 하나요?

**A**: 색상의 **개별** 자산보다 **자산군 변화**에 집중:
- 붉은 계열 위쪽(주식) 줄어들면 → 경보 반응 중
- 파란·노랑 계열 늘어나면 → 안전자산 이동

### ❓ Q3. 시각화 3에서 Top 10이 다 비슷해 보여요.

**A**: 거시적으로는 맞음. 미세 차이:
- 중립형·적극형은 주식 비중이 두꺼움
- M3는 M1 대비 약간 더 요철 있음
- Config C는 B보다 경보 발동 더 많음 → 반응 더 빈번

### ❓ Q4. 경보 L2+일 때도 주식이 완전 사라지진 않는데 왜?

**A**: EQUITY_CUT 테이블 기준:
- 보수형 L2 → 주식 35% **감축** (100% 제거 아님)
- 보수형 L3 → 주식 60% 감축
- 일부 주식은 항상 유지하여 **반등 기회** 보존

### ❓ Q5. 시각화 6에서 M3가 M1과 정말 차이 없나요?

**A**: 있으나 **작음**:
- Σ 전환 시점(보라 경계)에 미세 비중 점프
- 그러나 주식 총비중 패턴은 거의 동일
- Step 10 통계 검정에서 M3-M1 차이 -0.04 (유의하지 않음)

### ❓ Q6. 이 시각화가 실전 투자에 어떤 가치?

**A**:
- **투명성**: "언제 무엇을 왜 샀는지" 추적 가능
- **모니터링**: 예상 비중과 실제 운용 비중 비교
- **신뢰 구축**: 경보·레짐 시스템이 실제 작동 확인
- **학습**: 역사 위기(COVID, 2022 긴축)에서 어떻게 반응했는지 복기

### ❓ Q7. 검증 2가 일부 실패했는데 문제 있나요?

**A**: 아니오. 이유:
- 5일 창 비교에서 drift·분기 경계가 겹치면 미세 위반 가능
- 전체 추세는 명확히 "경보 상승 → 주식 감소"
- 엄격한 검증 기준의 한계이지 로직 오류 아님
