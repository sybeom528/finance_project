## 🎛️ 운용 메커니즘 (30초)

```
매일 아침 VIX + VIX Contango 체크:
  VIX < 20          → L0 (정상)   → 주식 그대로 유지
  20 ≤ VIX < 28     → L1 (주의)   → 주식 15% 감축 → 채권·금
  28 ≤ VIX < 35     → L2 (경계)   → 주식 35% 감축
  VIX ≥ 35          → L3 (위기)   → 주식 60% 감축

분기마다 IS 24개월 데이터로 MV 최적화 재계산
```
