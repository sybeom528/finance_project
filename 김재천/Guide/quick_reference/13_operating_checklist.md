       - 30자산 일별 가격
       - 기간: 최근 2년
[ ] 2. 로그 수익률 계산:
       log_ret = ln(P_t / P_{t-1})
[ ] 3. 기대수익률 μ 계산:
       μ = log_ret.mean() × 252
[ ] 4. 공분산 Σ 계산 (Ledoit-Wolf):
       from sklearn.covariance import LedoitWolf

## 🚨 이상 징후 경보 (즉시 대응 필요)

### ⚠️ 월간 이상 신호

| 신호 | 예상 원인 | 대응 |
|------|--------|------|
| Turnover > 2.0 | 경보 과빈번 | 최소 유지 기간 3일 적용 검토 |
| 월 거래비용 > 0.3% | 리밸런싱 과다 | 임계값 필터 추가 |
| 월 손실 > 5% | 전략 실패 or 위기 | 즉시 원인 분석 |
| 경보 0회 | 이상 저변동 기간 | 정상 (대응 불필요) |

| 신호 | 대응 |
|------|------|
| Sharpe < 0.5 (3개월 이상) | 전략 재검토 |

### 일일 스크립트 예시 (`daily_check.py`)

```python
import yfinance as yf
from datetime import datetime

def daily_alert():
    # VIX 데이터
    vix = yf.download('^VIX', period='5d')['Close'].iloc[-1]
    vix3m = yf.download('^VIX3M', period='5d')['Close'].iloc[-1]
    contango = vix3m - vix

    # 경보 판정
    if vix >= 35: level = 3
    elif vix >= 28: level = 2
    elif vix >= 20: level = 1
    else: level = 0
    if contango < 0:
        level = min(level + 1, 3)

    # 감축률
    cuts = {0: 0.00, 1: 0.15, 2: 0.35, 3: 0.60}

    return {
        'date': datetime.now().strftime('%Y-%m-%d'),
        'vix': round(float(vix), 2),
        'vix3m': round(float(vix3m), 2),
        'contango': round(float(contango), 2),
        'alert_level': level,
        'equity_cut_pct': cuts[level] * 100,
    }

if __name__ == '__main__':
    result = daily_alert()
    print('=' * 50)
    for k, v in result.items():
        print(f'  {k}: {v}')
    print('=' * 50)
    if result['alert_level'] >= 1:
        print('⚠️ 주식 감축 필요!')
```

## 📞 문제 해결

### 자주 발생하는 상황

**Q. 경보 발동 직후 시장이 반등하네?**
→ Whipsaw. 규칙 준수. 장기 통계상 전체 우위.

**Q. 분기 리밸런싱 시 turnover가 너무 큼?**
→ IS 데이터 이상 or γ 설정 점검. 정상 범위: 5~25%.

**Q. 여행 중 경보 체크 못 했는데?**
→ 복귀 즉시 판정 후 행동. 1~3일 지연은 허용.

**Q. 특정 자산 거래 체결 안 됨?**
→ 유동성 낮은 시점. 다음 날 재시도 또는 유사 자산(AGG 대신 BND) 대체.
