## ⚠️ 유의사항

### 🔻 거래비용
- 편도 15bps 가정 (실제 소매 환경 고려)
- 월평균 10~15회 리밸런싱 → 연 1~2% 비용
- 비용 30bps 이상이면 효과 반감

### 🔻 세금
- 미국 주식 배당소득세 15%
- 양도소득세 (한국 해외주식 22%)
- **세금 고려 시 Sharpe가 0.8~0.9로 하락** 가능

- 규칙 신뢰 + 감정 배제 필수
- 최소 3년 운용해야 통계적 의미

## 🧮 계산 도구 (Python 스니펫)

```python
def decide_action(vix_level, vix_contango, current_equity):
    """경보 레벨별 행동 결정 (Config B, 보수형)"""
    # 경보 레벨
    if vix_level >= 35: level = 3
    elif vix_level >= 28: level = 2
    elif vix_level >= 20: level = 1
    else: level = 0
    if vix_contango < 0:
        level = min(level + 1, 3)

    # 감축률
    cuts = {0: 0.00, 1: 0.15, 2: 0.35, 3: 0.60}
    cut = cuts[level]
    reduction = current_equity * cut

    return {
        'alert_level': level,
        'equity_reduction_pct': cut * 100,
        'equity_reduction_abs': reduction,
        'move_to_bonds': reduction * 0.70,
        'move_to_gold': reduction * 0.30,
        'new_equity': current_equity - reduction,
    }

# 사용 예시
result = decide_action(vix_level=32, vix_contango=-0.02, current_equity=0.40)
print(result)
# {'alert_level': 3, 'equity_reduction_pct': 60.0, ...}
```
