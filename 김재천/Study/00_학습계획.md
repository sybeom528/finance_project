## 학습의 큰 그림

현재 프로젝트는 다음 구조입니다.

```
입력: X_is.shape = (2310, 17)   ← 행마다 독립된 17개 피처
       ↓
모델: XGBoost                   ← 행 단위 학습 (날짜 순서 무시)
       ↓
출력: 5분위 확률 (proba[5])
       ↓
B-L: Q, Ω 추출 → 최적 포트폴리오
```

목표는 **모델만** 교체하는 것입니다.

```
입력: X_is.shape = (N, T, 17)    ← T일치 시퀀스를 묶어 입력
       ↓
모델: GRU/LSTM                   ← 시간 의존성 학습
       ↓
출력: 5분위 확률 (proba[5])      ← 동일
       ↓
B-L: Q, Ω 추출 → 최적 포트폴리오  ← 코드 수정 없음
```

**핵심:** 모델 입출력 인터페이스는 똑같이 유지하면서 내부만 바꾼다.
