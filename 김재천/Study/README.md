# 시계열 데이터 분석 학습 폴더

> Black-Litterman 포트폴리오 프로젝트에서 XGBoost를 시퀀스 모델(GRU/LSTM)로 교체하기 위한 학습 공간입니다.

## 폴더 구조

```
Study/
├── README.md                       # (이 파일) 학습 폴더 사용법
├── 00_학습계획.md                   # 전체 커리큘럼 + 진행 체크리스트
│
├── week1_시계열기초/                # 1주차: 정상성, 자기상관 등 이론
│   ├── 01_시계열데이터의본질.md
│   ├── 02_정상성.md
│   ├── 03_자기상관.md
│   └── 04_시계열분해.md
│
├── week2_시퀀스데이터/              # 2주차: PyTorch Dataset 만들기
│
├── week3_시퀀스모델/                # 3주차: GRU/LSTM 구현
│
└── week4_프로젝트통합/              # 4주차: 기존 코드에 통합
```

## 학습 자료 사용 규칙

### 자료 종류
- **`.md` 파일**: 이론 설명, 수식, 직관적 비유. 머리로 이해하는 단계.
- **`.ipynb` 파일**: 실제 코드를 돌려보며 손으로 익히는 단계.
- 한 토픽 = 이론 `.md` + 실습 `.ipynb` 조합.

### 학습 순서
1. `.md` 파일을 처음부터 끝까지 읽기 (개념 이해)
2. `.ipynb`를 셀 단위로 직접 실행 (코드 손에 익히기)
3. 마지막 "스스로 점검" 질문에 답해보기
4. 막히면 → 클로드에게 그 부분을 다시 설명 요청

### 진행 추적
- `00_학습계획.md`의 체크리스트를 매번 업데이트
- 각 토픽 완료 시 `[ ]` → `[x]` 로 변경

## 핵심 원칙

> "왜 그렇게 하는가"를 항상 먼저 이해한다.
> 코드는 그 다음이다.

이 폴더의 모든 자료는 *현재 진행 중인 Black-Litterman 프로젝트* 와 연결되도록 작성됩니다. 추상적인 toy example이 아니라, MSFT 등 실제 종목 데이터를 사용합니다.
