## 4. 근본 원인 분석

### 4-1. VIX가 오히려 성능을 악화시킨 이유

| 요인 | 설명 |
|---|---|
| **VIX-수익률 관계의 비선형·비정상성** | 고VIX → 하락이 일반적이나, 고VIX 이후 반등(리스크 프리미엄 수익) 도 빈번. 21일 선행 예측에 VIX는 일관된 방향 신호가 아님 |
| **적은 훈련 샘플 (134개/fold)** | VIX 레짐(저변동·고변동)별로 수익률 방향이 달라지는데, 134샘플로는 이 관계를 안정적으로 학습 불가 |
| **조기 종료가 너무 이름** | best_epoch mean = 5.18 (SPY), 5 epoch만에 중단 → 학습 자체가 불충분 |
| **VIX 극단값 효과** | COVID 2020·2022 연준 긴축 등 VIX 스파이크 구간이 소수 폴드에 집중 → 해당 폴드에서 스케일러가 왜곡 |
| **fold 14 집중 붕괴** | SPY R²=-131, QQQ R²=-250. 같은 폴드(14)에서 두 자산 동시 붕괴 → 해당 구간 특유의 VIX 패턴에 과적합 |

### 4-2. 왜 피처 없는 3차 Run이 상대적으로 나은가

피처 없는 모델(input=1)은 log_return만 보고 단순 평균 회귀(mean-reversion)에 가까운 예측을 한다.
예측값 분산이 낮아 R²_OOS는 음수지만, **방향이 크게 반전되는 극단 폴드가 드물다.**
VIX를 추가하면 모델이 VIX 방향 신호에 과도하게 반응하면서 잘못된 극단 예측을 만들어낸다.

## 5. Phase 1 현재 상태 진단

5번의 Run을 통해 다음이 확인되었다:

| 시도 | 결과 |
|---|---|
| hidden 축소 (128→32) | 소폭 악화. 근본 해결 안 됨 |
| Y_trailing (모멘텀 피처) | 대폭 악화. 작은 폴드서 역방향 과적합 |
| VIX (레짐 지표) | 대폭 악화. VIX-수익률 관계 불안정 |

**공통 결론**: 현재 설정(n_train=134, OOS=21, step=21)에서 외부 피처를 추가하면 모델이 불안정해진다.
root cause는 **훈련 샘플 수 절대 부족**이다.

## 6. 다음 단계 후보

### Option A — Phase 1 결과 수용 (3차 Run을 최종 출력으로)

3차 Run의 SPY hit_rate=0.6313, R²=-0.2118을 Phase 1 최종 결과로 확정한다.
- R²는 음수지만 hit_rate > 0.55로 방향성 신호는 있음
- Black-Litterman에서 Q(뷰) 크기를 작게 잡고 Omega(불확실성)를 높게 설정하면
  약한 신호도 활용 가능

| 장점 | 단점 |
|---|---|
| 즉시 Phase 2(Black-Litterman)로 진행 가능 | R²_OOS < 0 — 예측 파워 증명 불가 |

### Option B — Walk-Forward 파라미터 조정 (훈련 샘플 확대)

현재 IS=231 → IS=504 (2년)로 늘리면 n_train=504−63=441샘플로 증가.
단, 폴드 수가 줄어들어 OOS 평가 신뢰도 하락.

| 장점 | 단점 |
|---|---|
| 근본 원인(샘플 부족) 직접 해결 | 폴드 수 감소, 피처 공학 재시도 필요 |

### Option C — 모델 교체 (GRU 또는 단순화)

GRU는 LSTM 대비 파라미터 수 약 25% 감소 → 134샘플 환경에서 오히려 안정적일 수 있음.
또는 선형 리지 회귀(Ridge) 로 단순화하여 과적합을 원천 차단.

| 장점 | 단점 |
|---|---|
| 용량 감소로 과적합 완화 | 모델 교체 = 코드 구조 변경 필요 |

**현 시점 권장**: 시간 제약을 고려하면 **Option A** (3차 Run 결과 수용 후 Phase 2 진행)가 현실적이다.
Phase 1 목적은 Black-Litterman 뷰 입력이고, 약한 hit_rate 신호로도 Q/Omega 구성이 가능하다.
