## 논의 기록 시 체크리스트

새 논의 문서 추가 시:
1. 파일명 `YYYY-MM-DD_주제.md` 규약 준수
2. 문서 상단에 작성자·작성일·배경 명시
3. **사실(What) + 근거(Why) + 대안(How) + 평가(Trade-off)** 4요소 모두 기록
4. 본 인덱스 표에 1줄 추가 (날짜·파일링크·주제·결론·후속 task)
5. 해당 논의가 코드·문서에 반영되면 `상태 색인` 의 "결론·반영 완료" 로 이동
