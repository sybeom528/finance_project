## 2. 8차 Run 변경 내용

### 2-1. 변경 요약

| 파라미터 | 7차 Run | 8차 Run | 근거 |
|---|---|---|---|
| `IS` | 504일 (2년) | **756일 (3년)** | 훈련 샘플 483→714개 (+47.8%) |
| `INPUT_SIZE` | 2 (log_ret + ret_3m) | **1 (log_ret 단독)** | 6차 Run 기준 복원 — IS 효과만 단독 검증 |
| `SEQ_LEN` | 21 | 21 | 동일 유지 |
| 폴드 수 | 92 | **~80** | IS 확대로 감소 (트레이드오프) |
| 훈련 시퀀스/폴드 | 483 | **714** | IS - seq_len - purge = 756 - 21 - 21 |
| val 시퀀스/폴드 | ~97 | **~143** | 714 × 0.2 |

그 외 하이퍼파라미터(hidden=32, num_layers=1, dropout=0.3, lr=1e-3, weight_decay=1e-3, max_epochs=30, patience=5)는 동일 유지.

### 2-2. IS 확대 근거 (504 → 756)

```
IS=504, seq_len=21, purge=21:  유효 시퀀스 = 504 − 21 − 21 = 462
IS=756, seq_len=21, purge=21:  유효 시퀀스 = 756 − 21 − 21 = 714  (+54.5%)
```

폴드 수 감소 이유:
```
IS=504: (2493 − 567) / 21 ≈ 92 폴드
IS=756: (2493 − 819) / 21 ≈ 80 폴드
```

**기대 효과**: 더 다양한 시장 레짐을 한 폴드 내에서 학습 → IS-OOS 레짐 이동 문제 완화.

**한계**: ACF≈0 구조 한계는 IS 확대로 해결 불가. R²_OOS가 -0.025에서 0 이상으로 넘어가는지 확인.
