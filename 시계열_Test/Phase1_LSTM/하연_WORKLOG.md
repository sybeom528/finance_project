**§11 Lookback T 결정 기준 (세 가지 통계 + 도메인)**:

1. **PACF cutoff**: 첫 비유의 lag → AR 선형 order (lookback 하한)
2. **제곱 수익률 Ljung-Box**: 변동성 자기상관 소멸 lag
3. **AR(p) AIC/BIC**: `AutoReg` p=1~60 탐색, 정보 기준 최적 p
4. **도메인 지식**: 126일 ≈ 반기 (어닝시즌 2회 주기)
5. **학습자료 권장**: T=60 근처 (T<20 의존성 부족, T>200 표본 급감)

**설계 결정**:
- 학습자료_주의사항.md §5.2 경고("p-value만으로 판단 X") → §9 Ljung-Box 마크다운 셀에 명시

#### 1. §6 위치 이동 → §3 직후 §4로 재배치

**변경 전**: §6(분석 기간 집중 통계)이 통계 검정 섹션(§7~§10) 사이에 위치

**판단 근거**: §6은 Walk-Forward 인프라 수치(영업일 수·fold 수)로 통계 검정과 무관. 데이터 개요 확인 후 곧바로 분포/검정 섹션으로 이어지는 흐름이 더 자연스럽고, §7→§8→§9→§10 통계 검정 체인의 맥락이 보존됨.

| 구 번호 | 신 번호 | 내용 |

| §4 | §5 | log-return 분포 |
| §5 | §6 | ACF / PACF |
| §6 | §4 | 분석 기간 집중 통계 (이동) |
| §7~§12 | §7~§12 | 변경 없음 |

#### 2. §5 히스토그램에 이상치 axvline 오버레이 추가

**변경 내용**: §3에서 검출한 ±5σ 이상치 날짜의 `log_return` 값을 히스토그램 x축에 주황 점선(`axvline`)으로 표시. 첫 번째 선만 범례 레이블(`±5σ 이상치`) 표시.

**판단 근거**: §3(이상치 탐지)와 §5(첨도 해석)가 텍스트로만 연결되어 있던 것을 시각적으로 직결. fat-tail의 생성 원인인 COVID 2020-03 / 2025-04 관세 충격 날짜가 분포의 어느 극단에 위치하는지 즉시 확인 가능.

#### 3. §6 ACF 해석 ↔ §9 Ljung-Box 연결 문장 추가

**변경 내용**: §6 해석 마지막에 blockquote 추가 — "개별 lag의 통계적 유의성은 §9 Ljung-Box 검정에서 lag 전체를 동시 검정(Q 통계량 + 종합 p-value)하여 보강합니다." §9 도입부 비교표의 컬럼명도 §5→§6으로 갱신.

**판단 근거**: §6에서 결론이 완결되면 §9가 반복처럼 느껴짐. 포워드 레퍼런스 한 줄로 두 섹션이 개별 lag 검정 → 종합 검정의 계층적 구조임을 명시.

#### 4. §11에 seq_len 민감도 분석 셀 추가

**변경 내용**: AR AIC/BIC 셀 직후에 `seq_len_candidates = [21, 63, 126, 252]`를 루프하여 `AR(p=seq_len)` AIC와 최적 p 대비 ΔAIC를 비교하는 셀 삽입.

**판단 근거**: §11 결론에서 seq_len=252 비교 실험을 권장했으나 근거가 워크로그 텍스트에만 있었음. 선형 AR 관점에서 ΔAIC > 0이면 긴 seq_len이 추가 파라미터 낭비임을 수치로 확인 가능.

**⚠️ 후속 작업 필요**: 실제 seq_len 검증(LSTM val loss 비교)은 `02_setting_A_daily21.ipynb` seq_len sensitivity 섹션에서 별도 수행 예정. 선형 AR AIC는 통계 프록시일 뿐이며, 최종 채택은 LSTM 학습 결과 기준으로 판단.
