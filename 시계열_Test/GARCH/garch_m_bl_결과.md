# GARCH-M + Black-Litterman 포트폴리오 최적화 결과 요약

## 1. 모델 구조

```
GARCH-M (2-step)  →  Q 벡터  →  Black-Litterman  →  MV 최적화  →  Walk-Forward 백테스트
```

| 구성 요소 | 방법 | 비고 |
|-----------|------|------|
| 기대수익률 추정 | GARCH-M (2-step OLS) | Step1: GARCH(1,1), Step2: OLS로 λ 추정 |
| λ 제약 | λ = max(0, λ) | 이론상 리스크 프리미엄은 양수 |
| 공분산 행렬 | CCC-GARCH | R(LedoitWolf 상관) × D(GARCH 끝 시점 σ) |
| BL τ | 1/T (T=IS 관측치 수) | He-Litterman 표준 방식 |
| BL Ω | diag(P·τΣ·P') | He-Litterman 비례 방식 |
| 유니버스 | IS별 S&P 500 시총 상위 30개 동적 선정 | Wikipedia 멤버십 + 실제 발행주식수 |
| 거래비용 | 편도 10bps | 리밸런싱 시점 첫날 차감 |
| Walk-Forward | IS 2년 / OOS 3개월 | 2018 ~ 2025 |

|------|-------------|----------------|

> 샤프비율 기준 **2.2배** 초과 성과. MDD는 벤치마크 대비 소폭 개선.

## 5. 한계점 및 향후 연구 방향

### 치명적 한계
| 한계 | 설명 |
|------|------|
| **거래비용 단순화** | 10bps 고정 가정. 실제 대형주 매매 시 시장충격비용(market impact) 미반영 |
| **2022 하락장 취약** | 집중 포트폴리오(소수 종목 대형 비중)가 하락장에서 리스크 확대 |

### 방법론적 한계
| 한계 | 설명 | 향후 방향 |
|------|------|---------|
| λ 불안정 | 단기 OOS에서 λ 부호/크기 변동 | Bayesian shrinkage 또는 패널 데이터 λ 추정 |
| 공분산 시점 | CCC 상관관계가 여전히 IS 전체 평균 | DCC-GARCH (다변량) 적용 |
| 공매도 제약 | 롱온리 최적화로 분산 제한 | 공매도 허용 or 제약 완화 민감도 분석 |
| Look-ahead bias | 유니버스 선정 시 생존편향 부분 잔존 | 상장폐지/편출 종목 데이터 보강 |
