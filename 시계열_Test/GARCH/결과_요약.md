### 결론

- **샤프비율·Calmar 모두 ① 기본 Sharpe가 1위** — MDD 개선 전략들이 수익률을 너무 많이 희생함
- MDD 개선에 효과 있는 전략: ② ⑤ (캡 적용 시 MDD -29~30%대)
- MDD 직접 공략 목적함수(⑦ Sortino, ⑧ Calmar)는 MDD 개선 효과 미미 (-36~38%)
- **근본 원인**: -38.6% MDD는 주로 2020년 코로나 시장 전체 폭락 → 롱온리 주식 포트폴리오에서 가중치 최적화만으로는 회피 한계. 레짐 감지/동적 현금 편입 등 상위 레이어 필요

## GARCH-M 진단 (전체 기간 기준 참조 피팅)

- 피팅 대상: S&P 500 시총 상위 30개 (2016~2025)
- λ ≥ 0 제약 후 수렴 비율: ~97% (walk-forward 내 29~30/30)
- STI (일부 초기 윈도우): 데이터 부족으로 피팅 실패 → 해당 윈도우 29/30 수렴

## 모델 설명

### GARCH-M 기대수익률 Q 도출
1. GARCH(1,1) 피팅 → 조건부 분산 h_t 추출
2. OLS: r_t = μ + λ·h_t + ε_t → (μ, λ) 추정
3. λ = max(0, β_1) 비음수 클리핑 (이론적 리스크 프리미엄 양수)
4. 1-step ahead: σ²_{T+1} 예측
5. Q = (μ + λ·σ²_{T+1}) / 100 × 252 (연율화)

### CCC-GARCH 공분산 (시점 정렬)
- R: LedoitWolf로 추정한 상관행렬 (IS 기간 전체 사용)
- D: diag(σ_1, ..., σ_n) — IS 마지막 시점의 GARCH 예측 σ
- Σ_CCC = D @ R @ D
- 공분산과 Q 벡터가 동일한 IS 마지막 시점 σ를 공유

### Black-Litterman 공식
- Π = λ_BL · Σ · w_mkt (시장 균형 수익률)
- τ = 1/T (He-Litterman 표준)
- Ω = diag(P · τΣ · P') (불확실성 τ에 비례)
- μ_BL = [(τΣ)⁻¹ + P'Ω⁻¹P]⁻¹ [(τΣ)⁻¹Π + P'Ω⁻¹Q]
- ※ He-Litterman Ω 설정 시 τ는 μ_BL에서 상쇄 (τ-invariant)
