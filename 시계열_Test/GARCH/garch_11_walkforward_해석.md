# GARCH(1,1) Walk-Forward 변동성 예측 해석

**대상**: SPY, QQQ | **기간**: 2016-01-01 ~ 2025-12-31 | **모형**: t-분포 GARCH(1,1)

---

## 1. 분석 목적과 구조

`garch_11_volatility.ipynb`의 in-sample 분석과 달리, 이 노트북은 **미래 데이터를 전혀 사용하지 않는 Walk-Forward** 방식으로 실제 예측 성능을 측정한다.

### Walk-Forward 구조

```
t=0          IS(최소 500일)          fold_start     fold_start+21
|────────────────────────────────────|──────────────────|
          GARCH 파라미터 추정               OOS 예측 구간
                                    (21일 후 step 이동 → 반복)
```

- **IS (In-Sample)**: 매 fold마다 확장 (expanding window) — 전체 96 fold
- **Step**: 21일 (Phase 1.5 HAR-RV와 동일)
- **예측 목표**: 1일 앞(§4) 와 21일 앞(§5) 두 종류

### 타겟 정의

| 예측 | 타겟 공식 | 비교 대상 |
|---|---|---|
| 1일 앞 | $\vert r_{t+1} \vert$ (절댓값 수익률) | 없음 (GARCH 단독) |
| 21일 앞 | $\log(\text{std}(r_{t+1:t+22},\, \text{ddof}=1))$ | HAR-RV (Phase 1.5) |

21일 앞 타겟은 Phase 1.5의 `build_daily_target_logrv_21d`와 **동일한 공식** → HAR-RV 결과와 직접 비교 가능.

## 2. §4 — 1일 앞 변동성 예측 결과

### 2-1. 예측 방식

매 fold에서 t-GARCH(1,1)을 IS로 적합한 뒤, OOS 21일 동안 **재귀적(recursive)** 으로 1-step 예측을 생성한다.

$$\hat{\sigma}^2_t = \hat{\omega} + \hat{\alpha}\,(r_{t-1}-\hat{\mu})^2 + \hat{\beta}\,\hat{\sigma}^2_{t-1}$$

OOS 구간에서 실현 수익률 $r_{t-1}$을 업데이트에 사용하되, 파라미터 $\hat{\omega}, \hat{\alpha}, \hat{\beta}$는 IS 데이터로만 추정 → **순수 1-step ahead, look-ahead 없음**.

### 2-2. 결과 수치

| | SPY | QQQ |
|---|---|---|
| OOS 예측 수 | 2,013일 | 2,013일 |
| **RMSE** | **0.008460** | **0.010607** |
| **QLIKE** | **1.5175** | **1.5156** |
| corr($\hat{\sigma}_t$, $\vert r_t \vert$) | 0.5287 | 0.4382 |
| RMSE (고변동성 구간) | 0.008942 | 0.010410 |
| RMSE (저변동성 구간) | 0.007950 | 0.010799 |

### 2-3. 해석

**[RMSE]**  
절댓값 수익률 $\vert r_t \vert$은 변동성의 noisy proxy이므로 RMSE 절댓값 자체보다 **방향 포착 여부**가 더 중요하다.  
QQQ RMSE > SPY RMSE는 QQQ의 구조적 고변동성(일평균 σ ≈ 1.41% vs 1.14%)을 그대로 반영한다.

**[QLIKE ≈ 1.52]**  
QLIKE = $\text{mean}\!\left(\frac{r_t^2}{\hat{\sigma}_t^2} - \log\frac{r_t^2}{\hat{\sigma}_t^2} - 1\right)$ 는 변동성 예측의 학술 표준 손실함수로, 0에 가까울수록 좋다.  
1.52는 완벽한 예측과 거리가 있으나, $\vert r_t \vert$가 당일 변동성의 noisy proxy이기 때문에 이론적으로도 0에 근접하기 어렵다. 완벽한 모형이라도 이 프록시 기준 QLIKE는 높게 나온다 (정보 한계).

> QLIKE 계산 시 $r_t = 0$인 날은 $\log(0)$ 방지를 위해 제외 (SPY 4일, QQQ 1일).

**[corr = SPY 0.53 / QQQ 0.44]**  
GARCH 예측 $\hat{\sigma}_t$와 실제 $\vert r_t \vert$의 Pearson 상관이 0.44~0.53으로 유의미하다.  
즉, GARCH는 **"언제 변동성이 높고 낮은지"** 방향 포착은 절반 이상 성공하고 있다.  
SPY corr > QQQ corr: QQQ는 α+β=1.000 (IGARCH 경계)로 과도한 지속성 때문에 방향 포착도 상대적으로 약하다.

**[고변동성 vs 저변동성 구간]**

| | SPY | QQQ |
|---|---|---|
| 고변동성 RMSE | 0.008942 (전체보다 높음) | 0.010410 (전체보다 낮음) |
| 저변동성 RMSE | 0.007950 (전체보다 낮음) | 0.010799 (전체보다 높음) |
| 패턴 | 충격 크기 과소추정 | 저변동성 구간에서 과대 추정 |

- **SPY**: 고변동성 구간에서 RMSE가 더 높음 → 충격의 방향은 맞히지만 크기를 과소추정 (이미 높은 σ 예측 + 실제는 더 큰 $\vert r \vert$)
- **QQQ**: 저변동성 구간에서 RMSE가 더 높음 → α+β≈1의 지속성 효과로 변동성이 내려온 구간에서도 GARCH 예측이 높게 형성되어 과대 추정

## 3. §5 — 21일 앞 변동성 예측 결과

### 3-1. 예측 방식

매 fold에서 t-GARCH(1,1)을 IS로 적합한 뒤, `forecast(horizon=21)`로 21일 분산 예측을 생성한다.

$$\hat{\sigma}^2_{t+k} = \hat{\omega}\sum_{j=0}^{k-1}(\hat{\alpha}+\hat{\beta})^j + (\hat{\alpha}+\hat{\beta})^k \hat{\sigma}^2_t, \quad k=1,\ldots,21$$

21일 평균 RV를 log 변환해 타겟과 비교:

$$\log\!\left(\sqrt{\frac{1}{21}\sum_{k=1}^{21}\hat{\sigma}^2_{t+k}}\right) \quad \text{vs} \quad \log(\text{std}(r_{t+1:t+22}))$$

### 3-2. 결과 수치

| | SPY | QQQ |
|---|---|---|
| OOS fold 수 | 95 | 95 |
| **GARCH RMSE** | **0.4214** | **0.3893** |
| **QLIKE** | **0.3718** | **0.2983** |
| corr(예측, 실제) | 0.6187 | 0.6096 |
| **편향 (bias = 예측 − 실제 평균)** | **+0.1361** | **+0.1218** |
| RMSE (고변동성 구간) | 0.4033 | 0.3628 |
| RMSE (저변동성 구간) | 0.4384 | 0.4136 |
| **HAR-RV RMSE** (Phase 1.5) | **0.3646** | **0.3308** |
| **ΔRMSE (GARCH − HAR-RV)** | **+0.0568** | **+0.0585** |

### 3-3. 해석

**[HAR-RV 우세]**  
두 종목 모두 GARCH가 HAR-RV보다 RMSE 기준으로 약 **15~18% 더 부정확**하다.

**[핵심 원인: 양의 편향(bias ≈ +0.12~0.14)]**  
GARCH 예측이 실제 log-RV보다 **체계적으로 과대 추정**된다.  
로그 스케일 +0.136은 실제 변동성 대비 약 $e^{0.136} - 1 \approx \mathbf{14.6\%}$ 높게 예측하는 것과 같다.

원인은 두 가지다:

| 원인 | 설명 |
|---|---|
| α+β ≈ 1 (지속성 과도) | 변동성 충격이 21일간 거의 감쇠 없이 유지 → 저변동성 진입 후에도 높은 예측값 |
| QQQ IGARCH 경계 | α+β=1.000 → 장기 분산 수렴 공식 $\omega/(1-\alpha-\beta)$ 분모=0 → 21일 예측이 현재 변동성 수준에 고착 |

**[corr ≈ 0.61 — 방향 포착은 유사]**  
HAR-RV와 GARCH 모두 비슷한 방향 상관(0.61)을 가진다.  
GARCH가 지는 이유는 방향 예측 실패가 아니라 **수준(level)의 과대 추정** 때문이다.  
bias를 보정(예측에서 평균 편향 차감)하면 RMSE 격차의 상당 부분을 줄일 수 있다.

**[고변동성 vs 저변동성 구간]**

| 구간 | SPY | QQQ | 해석 |
|---|---|---|---|
| 고변동성 (COVID, 금리인상) | RMSE 0.4033 | 0.3628 | GARCH 상대적으로 선전 — 충격 시 높은 예측값이 실제와 가까워짐 |
| 저변동성 (2021 안정기) | RMSE 0.4384 | 0.4136 | GARCH 더 크게 실패 — 지속성 효과로 변동성 하락을 늦게 반영 |

## 4. §6 — HAR-RV vs GARCH 종합 비교

### 4-1. 21일 앞 RMSE 직접 비교

| 모델 | SPY RMSE | QQQ RMSE | 비고 |
|---|---|---|---|
| **HAR-RV** (Phase 1.5) | **0.3646** | **0.3308** | **승자** |
| t-GARCH(1,1) | 0.4214 | 0.3893 | ΔRMSE ≈ +0.057~0.059 |

### 4-2. 산점도 해석 (실제 log-RV vs 예측 log-RV)

완벽한 예측이면 모든 점이 y=x 대각선 위에 분포해야 한다.  
GARCH 산점도에서는 점군이 대각선보다 **위쪽에 치우쳐** 있어 양의 편향(과대 추정)을 시각적으로 확인할 수 있다.

| 시각적 패턴 | 의미 |
|---|---|
| 점군이 y=x 위쪽 | GARCH가 실제보다 변동성을 체계적으로 높게 예측 |
| 양 극단에서 분산 확대 | 고변동·저변동 구간 모두 오차 큼 (bias + 추가 분산) |
| 기울기 방향은 맞음 (corr=0.61) | 수준만 틀렸을 뿐, 방향 포착은 유효 |

### 4-3. 두 모형의 역할 분담

| 관점 | HAR-RV | t-GARCH(1,1) |
|---|---|---|
| **21일 앞 RMSE** | ✅ 우수 (0.33~0.36) | ❌ 열세 (+0.057~0.059) |
| **1일 앞 예측** | ❌ 미지원 (설계상) | ✅ corr=0.44~0.53 |
| **편향** | 낮음 (직접 RV 회귀) | 높음 (+0.12~0.14, IGARCH 효과) |
| **해석 가능성** | 일·주·월 RV 기여도 직관적 | α, β 파라미터 — 경제적 의미 명확 |
| **주요 강점** | 21일 평균 수준 자연 포착 | 단기 변동성 클러스터링 포착 |

## 5. 결론 및 다음 단계

### 5-1. 전체 성능 요약

| 예측 | 모델 | SPY | QQQ | 비고 |
|---|---|---|---|---|
| 1일 앞 | t-GARCH(1,1) | RMSE=0.00846, corr=0.53 | RMSE=0.01061, corr=0.44 | 비교 대상 없음 |
| 21일 앞 | **HAR-RV** | **RMSE=0.3646** | **RMSE=0.3308** | **승자** |
| 21일 앞 | t-GARCH(1,1) | RMSE=0.4214 | RMSE=0.3893 | ΔRMSE ≈ +0.057~0.059 |

### 5-2. 왜 HAR-RV가 이겼나

GARCH가 HAR-RV에 진 이유는 방향 예측 실패가 아니라 **수준 과대 추정(bias ≈ +0.13)**이다.

| 원인 | 설명 |
|---|---|
| α+β ≈ 1 (지속성 과도) | 변동성 충격을 21일간 거의 감쇠 없이 유지 → 저변동성 구간 진입 후에도 높은 예측값 |
| QQQ IGARCH 경계 | α+β=1.000 → 장기 평균 분산 미수렴 → 21일 예측이 현재 분산 수준에 고착 |
| HAR-RV의 구조적 강점 | 일간·주간·월간 RV를 직접 회귀 → 21일 평균 수준을 자연스럽게 반영, 편향 적음 |

### 5-3. 1일 앞 예측의 의미

1-step ahead corr=0.44~0.53은 **GARCH가 변동성 방향을 절반 이상 맞힌다**는 의미다.  
HAR-RV는 21일 집계 단위로 설계되어 1일 앞 비교가 불가 → **GARCH만 유일하게 제공하는 단기 신호**.  
단기 예측에서는 GARCH의 재귀적 구조가 변동성 클러스터링을 효과적으로 포착한다.

### 5-4. 다음 단계 제안

| 개선 방향 | 기대 효과 |
|---|---|
| **편향 보정**: OOS IS 구간 평균 bias 차감 | ΔRMSE 0.057의 상당 부분 해소 가능 |
| **GJR-GARCH**: 하락 비대칭 포착 (`o=1`) | 잔차 왜도 −0.86/−0.68 처리 → 고변동성 구간 RMSE 개선 |
| **GARCH 조건부 변동성을 HAR-RV 피처로 추가** | 두 모형의 강점 결합 |
| **표본 기간 재검토** | QQQ α+β=1.000 (IGARCH) — 2010~2025 등 기간 변경으로 재확인 |
