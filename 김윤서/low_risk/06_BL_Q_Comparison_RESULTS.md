## 핵심 발견

### 발견 1: Q_lambda가 Sharpe·MDD 동시 1위 (Sharpe 1.112, MDD -11.84%)

5가지 Q 방법 중 Q_lambda가 Sharpe와 MDD 모두에서 가장 우수하다.

**Q_lambda 공식**: `Q = Q_FIXED × clip(λ_t / λ_mean, 0.1, 3.0)`
- `λ_t`: 학습 기간(60개월) SPY 초과수익 / SPY 분산으로 추정한 시장 위험회피계수
- `λ_mean = 2.5`: BL 문헌 장기 평균값 (He & Litterman 2002)
- scale 범위 [0.1, 3.0]으로 극단값 방지

**작동 원리**:
- 시장이 안정적(고수익·저변동성 → λ↑) → Q 강화 → 저위험 선호를 강하게 반영
- 시장이 불안(저수익·고변동성 → λ↓) → Q 약화 → 포트폴리오 보수적으로 유지

자연스럽게 강세장에서는 저위험 tilt를 강하게, 위기 때는 약하게 가져가면서 변동성을 줄이고 Sharpe를 높인다.

**실제 λ 통계**: 평균 ≈ 2.5, 범위 0.5~10.0으로 클리핑

---

### 발견 2: Q_FIXED — 견고한 차선 (Sharpe 1.074)

Q_lambda 다음으로 Q_FIXED=0.003이 Sharpe 1.074로 우수하다.

**메커니즘**: `diff = Q - P·π`에서 Q=0.003은 작은 값이라 BL 조정이 CAPM에서 소폭만 이탈.  
과도한 집중을 막는 완충 역할로 분산을 유지하면서 저위험 tilt를 부드럽게 인가한다.

07_Q_Sensitivity에서도 GARCH vol 기준으로 Q=0.003이 최적값으로 확인됨 (Sharpe 1.019).  
Q_lambda를 쓰기 어려운 경우(단순성 선호) Q_FIXED가 최선.

---

### 발견 3: Q_hist — 누적수익률은 높지만 Sharpe 열세 (Sharpe 0.896)

```
Q_hist:  누적수익률 613.83%, Sharpe 0.896, 연환산변동성 14.24%
Q_FIXED: 누적수익률 608.29%, Sharpe 1.074, 연환산변동성 11.44%
```

60개월 평균 스프레드를 Q로 쓰면 Q 수준이 Q_FIXED보다 크게 나오는 달이 많아 BL이 과집중을 유도한다.  
수익은 더 올리지만 변동성이 급등해 Sharpe가 낮아진다.

---

### 발견 4: Q_momentum — 역방향 신호 문제 (Sharpe 0.597, MDD -42.00%)

최근 12개월만 보는 단기 추세는 저위험 프리미엄이 역전된 달에 Q를 음수로 만든다.  
음수 Q → BL이 고위험 쪽으로 기울어짐 → 저위험 전략과 반대 방향 → MDD -42% 폭발.

Q_hist(60M)는 장기 평균이라 거의 항상 양수인 반면,  
Q_momentum(12M)은 최근 추세를 따라 음수로도 쉽게 뒤집힌다.

---

### 발견 5: Q_ff3 — 최하위 (Sharpe 0.586, MDD -51.48%)

FF3 팩터(시장, SMB-소형주, HML-가치주)는 **저변동성 이상현상을 직접 설명하는 팩터가 아니다**.  
FF3 회귀로 예측한 Q는 월별 변동이 극심하고 음수가 빈번해 BL을 왜곡한다.  
수익률 예측 자체의 어려움(FF3로 설명 못하는 변동 大)이 근본 원인.

---

### 발견 6: BL 프레임워크 자체는 유효

```
Q_lambda:  Sharpe 1.112  │
Q_FIXED:   Sharpe 1.074  │  → BL이 CAPM(0.882)·SPY(0.898) 모두 초과
CAPM:      Sharpe 0.882  │
SPY:       Sharpe 0.898  │
```

Q를 잘 설정하면 BL은 순수 CAPM보다 Sharpe를 명확히 개선한다.  
저위험 이상현상이 이 유니버스(2011~2025)에서 실증적으로 유효함을 확인.
