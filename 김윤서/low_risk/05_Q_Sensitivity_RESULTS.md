## 핵심 발견

### 발견 1: Q=0.001~0.003 구간이 Sharpe 고원

```
Q=0.001: Sharpe 1.001
Q=0.002: Sharpe 1.018
Q=0.003: Sharpe 1.019  ← 최적
Q=0.005: Sharpe 0.986  ← 급격 하락 시작
```

Q=0.001~0.003 구간은 Sharpe가 1.001~1.019로 평탄하다.  
이 구간에서는 Q 값이 다소 달라도 성과 차이가 미미하므로, Q=0.003은 **로버스트한 선택**이다.

---

### 발견 2: Q 증가 → Sharpe 단조 감소 (0.003 이후)

```
Q 증가에 따른 변화:
  수익률: 13.67% → 10.46%  (Q=0.001→0.020, 꾸준히 하락)
  변동성: 12.21% → 12.47%  (U자형: 0.005에서 최저, 이후 상승)
  Sharpe: 1.001  →  0.721  (꾸준히 하락)
  MDD:   -17.33% → -26.13% (Q=0.005 최저점, 이후 악화)
```

**경제적 해석**:

BL 조정 크기는 `diff = Q - Pπ`에 비례한다.  
Q가 클수록 BL이 저위험 종목을 더 강하게 over-weight → 포트폴리오 집중도 상승.

| Q 구간 | BL 조정 강도 | 포트폴리오 성격 |
|--------|------------|----------------|
| 0.001~0.003 | 약함 | CAPM에서 소폭 이탈, 분산 유지 → Sharpe 최고 |
| 0.005~0.007 | 중간 | 저위험 tilting 가시적, 수익률·변동성 균형 붕괴 시작 |
| 0.010~0.020 | 강함 | 과도한 집중, 수익률 하락 + 변동성 상승 → Sharpe 폭락 |

---

### 발견 3: Q=0.005부터 CAPM Sharpe(0.882)에 근접

```
Q=0.010: Sharpe 0.893  ← CAPM(0.882)과 거의 같음
Q=0.015: Sharpe 0.798  ← CAPM보다 낮음
Q=0.020: Sharpe 0.721  ← SPY(0.898)보다도 낮음
```

Q가 너무 크면 BL 뷰가 오히려 포트폴리오를 왜곡한다.  
Q=0.010은 BL을 쓰는 의미가 거의 없고, Q=0.015 이상은 BL이 역효과를 낸다.

---

### 발견 4: MDD는 Q=0.005에서 최저(-16.16%), 이후 악화

```
Q=0.003: MDD -16.73%
Q=0.005: MDD -16.16%  ← MDD 최소점
Q=0.010: MDD -17.51%
Q=0.015: MDD -22.97%
Q=0.020: MDD -26.13%
```

MDD 관점에서는 Q=0.005가 최적이지만 Sharpe가 0.986으로 낮아진다.  
Q=0.003은 MDD(-16.73%)와 Sharpe(1.019) 모두에서 균형이 가장 좋은 지점이다.

---

## Q 값과 BL 공식의 연결

$$\mu_{BL} = \pi + \tau\Sigma P^T (P\tau\Sigma P^T + \Omega)^{-1} \underbrace{(Q - P\pi)}_{\text{조정 크기}}$$

- `Pπ` ≈ 음수 (CAPM은 고위험 선호)
- `Q - Pπ` = Q + |Pπ| → Q가 클수록 조정 크기 ↑
- 조정이 과도해지면 저위험 종목에 극단적 집중 → 분산 효과 손실

Q=0.003이 최적인 이유: 저위험 이상현상에 대한 **"충분히 작은 신호"**로 포트폴리오를 CAPM에서 살짝만 밀어내면서 분산을 유지.
