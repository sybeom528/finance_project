### 분석 1: Rank IC 비교 (셀 1)

```
                      GARCH     Baseline  차이(G-B)
평균 IC                 0.5875    0.5680    +0.0194
중앙값 IC                0.5855    0.5677    +0.0178
ICIR                  7.2803    5.1041
양수 비율                 100.0%     100.0%
GARCH 우위 월            104개 / 180개월
GARCH 우위 비율           57.8%
```

**`평균 IC: GARCH 0.5875, Baseline 0.5680`**  
- IC(Information Coefficient)는 예측 vol 순위와 실제 vol 순위 간의 Spearman 상관계수
- IC 1.0 = 완벽하게 순위 예측, IC 0.0 = 랜덤과 같음, IC -1.0 = 완벽하게 반대
- **0.5875는 매우 높은 수치**: 금융 예측에서 IC 0.05만 돼도 "유의미하다"고 본다. 0.59는 예측이 매우 강하다는 의미
- 이 높은 IC는 변동성(vol)이 수익률보다 예측 가능한 특성 덕분 — 변동성 클러스터링(ARCH 효과) 때문에 현재 변동성이 높으면 다음 달도 높을 가능성이 높다
- GARCH가 Baseline보다 +0.019 높다는 것은, GARCH가 현재 vol을 단순히 이월하는 것보다 미래 vol을 더 정확하게 예측한다는 의미

**`중앙값 IC: 0.5855 vs 0.5677`**  
- 평균(0.5875)과 중앙값(0.5855)이 거의 같다 → IC 분포가 좌우 대칭 (극단적 아웃라이어 없음)
- 평균 ≈ 중앙값이면 안정적 예측 성능을 의미. 평균 >> 중앙값이면 일부 달에만 성능이 몰려 있다는 신호

**`ICIR: 7.28 vs 5.10`**  
- ICIR = IC의 평균 ÷ IC의 표준편차 = 정보비율(Information Ratio)의 IC 버전
- 값이 클수록 IC가 안정적으로 나온다는 뜻 (산발적이 아니라 매달 일관되게 좋음)
- ICIR > 2.0이면 통계적으로 유의미한 예측력으로 간주
- GARCH 7.28 vs Baseline 5.10 — 두 방법 모두 매우 강하지만 GARCH가 43% 더 안정적
- GARCH의 IC가 더 일관성 있다는 뜻 → 예측 품질이 특정 시기에만 좋은 게 아니라 전 기간에 걸쳐 지속됨

**`양수 비율: 100.0% 양쪽 다`**  
- 180개월 중 단 한 달도 IC가 음수(예측 방향이 반대)인 달이 없었다는 뜻
- 완벽한 방향 일치 — 항상 "vol이 높다고 예측한 종목이 실제로도 높았다"
- 이것은 GARCH와 Baseline 모두에 해당. 변동성 자체가 예측 가능한 신호임을 확인

**`GARCH 우위 월: 104개/180개월 (57.8%)`**  
- 매달 GARCH IC와 Baseline IC를 비교했을 때, GARCH가 더 높았던 달이 104개
- 57.8%의 달에서 GARCH 승 → 과반수이지만 압도적이지 않음
- 나머지 42.2%의 달에서는 Baseline이 더 정확했음 (특히 고변동성 구간: 발견 3 참조)

### 분석 2: P 행렬 일치율 (셀 2)

```
저위험 그룹 평균 일치율:  58.9%
고위험 그룹 평균 일치율:  62.3%
전체 평균 일치율:         60.6%
일치율 90% 이상인 월:     0개 / 180개월
일치율 50% 미만인 월:     2개월
```

**이 분석의 목적**:  
GARCH가 만드는 P 행렬(저위험/고위험 종목 선택)이 단순 Baseline과 얼마나 다른지 측정.  
만약 두 방법이 같은 종목을 고른다면, GARCH를 쓰는 의미가 없다.

**`저위험 그룹 평균 일치율 58.9%`**  
- 매월 GARCH가 저위험으로 분류한 종목의 30% 그룹 중, Baseline도 저위험으로 분류한 종목 비율이 평균 58.9%
- 반대로 말하면, **매달 GARCH 저위험 그룹의 41.1%는 Baseline과 다른 종목**
- "저위험 30% 그룹"이 610종목 기준 약 183종목이라면, 매달 약 75종목을 GARCH만 다르게 선택

**`고위험 그룹 평균 일치율 62.3%`**  
- 고위험 그룹 일치율이 저위험보다 약간 높음 (62.3% vs 58.9%)
- 고위험 종목은 일반적으로 변동성이 크고 명확해서 두 방법 모두 비슷하게 지목하는 경향
- 저위험 그룹은 "중간 정도 변동성을 가진 종목 중 누구를 선택할지"가 더 애매해서 방법 간 차이가 큼

**`전체 평균 일치율 60.6%`**  
- 핵심 수치: GARCH와 Baseline이 동일한 분류를 내리는 비율이 60.6%
- **39.4%의 종목은 두 방법이 다르게 분류** → GARCH는 단순 이월과 구별되는 자체 분류 논리를 갖고 있음
- 만약 일치율이 95% 이상이었다면 "GARCH를 쓰나 Baseline을 쓰나 거의 같다"는 결론이 나왔겠지만, 60%는 실질적 차별화를 의미

**`일치율 90% 이상인 월: 0개/180개월`**  
- 180개월 중 단 한 달도 두 방법이 90% 이상 같은 종목을 선택하지 않았다
- GARCH는 매달 Baseline과 의미 있게 다른 P를 만들어냄 → 독립적인 정보를 제공함

**`일치율 50% 미만인 월: 2개월`**  
- 극단적으로 다른 경우(절반도 안 일치)는 2달뿐
- 대부분의 달은 두 방법 사이에 어느 정도 겹침이 있음 (완전히 다른 포트폴리오는 아님)

### 분석 3: P 분류 정확도 — Precision (셀 3)

```
                    GARCH     Baseline  차이(G-B)
저위험 Precision:   0.571     0.546     +0.025
고위험 Precision:   0.598     0.580     +0.018
평균 Precision:     0.584     0.563     +0.021

[참고] 랜덤 분류 기대값: 30% (PCT_GROUP = 0.3)
```

**이 분석의 목적**:  
"실제로 다음 달에 저위험이었던 종목"을 얼마나 맞혔는지(Precision) 측정.  
IC가 높아도 분류 경계에서 틀리면 P 행렬이 잘못될 수 있어, 별도로 확인한다.

**Precision 정의**:
$$\text{Precision}_{\text{low}} = \frac{\text{GARCH가 저위험으로 분류했는데 실제로도 다음 달 저위험이었던 종목 수}}{\text{GARCH가 저위험으로 분류한 종목 수}}$$

**`저위험 Precision: GARCH 0.571, Baseline 0.546`**  
- GARCH가 "이 종목은 다음 달 저위험이다"라고 분류한 것의 57.1%가 실제로 맞았다
- Baseline은 54.6% 적중
- 랜덤으로 분류하면 기대 Precision = 30% (PCT_GROUP=0.30이니까)
- 두 방법 모두 30%보다 약 27%p 높음 → 실질적 예측 정보가 있음을 확인
- GARCH가 추가로 +2.5%p: 100종목 중 약 2.5종목을 더 정확하게 분류

**`고위험 Precision: GARCH 0.598, Baseline 0.580`**  
- 고위험 분류가 저위험보다 Precision이 더 높음 (0.598 vs 0.571)
- 고위험 종목은 상대적으로 명확한 신호(높은 현재 변동성)를 가져 맞추기 쉬움
- 저위험 종목이 더 어려운 이유: 변동성이 낮은 종목들 사이에서 다음 달에도 낮을 것을 선별하는 게 더 까다로움

**`평균 Precision: GARCH 0.584, Baseline 0.563`**  
- 전반적으로 GARCH가 실제 저위험/고위험 종목을 2.1%p 더 잘 맞힘
- 이 2.1%p 차이가 BL 포트폴리오로 전달될 때는 발견 6(팩터 수익률)에서 확인되듯 0.013%/월 손실 절감으로 나타남

> **주의**: PCT_GROUP=0.30이므로 예측 그룹 크기 = 실제 그룹 크기 = n_group  
> 이 경우 수학적으로 Precision = Recall = F1이 성립.  
> 따라서 별도로 Recall을 계산할 필요가 없으며, Precision 하나만으로 분류 품질을 완전하게 평가할 수 있다.

### 분석 3_5: 변동성 분리도 & P 팩터 수익률 (셀 new-sep)

#### ① 변동성 분리도

```
                         GARCH       Baseline
예측 저위험 그룹 평균 vol:  0.2058      0.2100
예측 고위험 그룹 평균 vol:  0.3549      0.3510
vol 비율 (low/high):        0.5759      0.5997   ← 낮을수록 분리 우수
vol 절대 갭 (high-low):     0.1490      0.1410
분리 성공률 (저<고 인 달):  100.0%       100.0%
```

**이 분석의 목적**:  
Precision은 "맞고 틀림"의 이진 판단. 그러나 실제로 중요한 것은 **"저위험 그룹의 실제 vol이 고위험보다 얼마나 낮은가"** — 즉 vol 공간에서의 분리 정도다.

**`예측 저위험 평균 vol: GARCH 0.2058, Baseline 0.2100`**  
- GARCH가 저위험으로 분류한 종목의 다음 달 실제 vol 평균이 0.2058 (연환산 약 20.6%)
- Baseline 저위험 그룹의 실제 vol은 0.2100 (약 21.0%)
- GARCH 저위험 그룹의 실제 vol이 Baseline보다 0.0042 더 낮다
- 즉, GARCH가 골라낸 저위험 종목이 실제로 더 조용한(낮은 변동성) 종목들이었다

**`예측 고위험 평균 vol: GARCH 0.3549, Baseline 0.3510`**  
- GARCH 고위험 그룹의 실제 vol이 0.3549로 Baseline(0.3510)보다 약간 높음
- GARCH가 고위험으로 분류한 종목이 실제로 더 위험한 종목이었다는 의미

**`vol 비율 (저/고): GARCH 0.5759, Baseline 0.5997`**  
- 저위험 그룹의 vol을 고위험 그룹의 vol로 나눈 비율
- 0.5759 = 저위험 그룹 vol이 고위험의 약 58% 수준 → 두 그룹이 42% 차이나는 분리
- Baseline은 0.5997 → 두 그룹이 40% 차이
- **비율이 낮을수록 더 날카롭게 분리**: GARCH(0.5759) < Baseline(0.5997) → GARCH 우위
- 이상적으로는 비율이 0에 가까울수록(저위험이 극도로 낮은 vol) 좋지만, 실제 시장에서 0.57~0.60도 강한 분리

**`vol 절대 갭 (고-저): GARCH 0.1490, Baseline 0.1410`**  
- 고위험 그룹과 저위험 그룹의 vol 차이 (절대값)
- GARCH: 0.3549 - 0.2058 = 0.1490 (약 14.9%p)
- Baseline: 0.3510 - 0.2100 = 0.1410 (약 14.1%p)
- GARCH가 만든 분류가 더 큰 vol 갭을 만들어냄 → BL에서 저위험 over-weight 효과가 더 분명하게 발현될 수 있음

**`분리 성공률: 100.0% 양쪽 다`**  
- 180개월 모든 달에서 "예측 저위험 그룹의 실제 vol < 예측 고위험 그룹의 실제 vol"이 성립
- 방향성이 단 한 번도 역전되지 않았다 — 두 방법 모두 저위험/고위험을 일관되게 구분

#### ② P 팩터 수익률

```
                      GARCH        Baseline
long(저위험) 평균 수익: +1.050%/월    +1.051%/월
short(고위험) 평균 수익:+1.291%/월    +1.305%/월
팩터 스프레드:          -0.241%/월    -0.254%/월
연환산 팩터 수익:       -2.89%/년    -3.05%/년
팩터 Sharpe:            -0.197        -0.213
양수 비율:              51.4%         48.6%
```

**이 분석의 목적**:  
분류가 경제적으로 유효한지 "long 저위험 - short 고위험" 포트폴리오의 실제 수익으로 검증.  
분류 Precision이 높아도 수익으로 연결되지 않으면 BL 뷰로서의 가치가 줄어든다.

**`long(저위험) 평균 수익: GARCH +1.050%/월, Baseline +1.051%/월`**  
- GARCH가 선택한 저위험 종목들의 다음 달 등비중 평균 수익률이 +1.050%/월
- 두 방법의 long 수익이 거의 동일 → 저위험 종목들의 수익률 자체는 비슷

**`short(고위험) 평균 수익: GARCH +1.291%/월, Baseline +1.305%/월`**  
- 고위험 종목들의 평균 수익이 +1.291%/월로 저위험(+1.050%)보다 높음
- 이것이 팩터 스프레드가 음수가 되는 직접 원인

**`팩터 스프레드: GARCH -0.241%/월, Baseline -0.254%/월`**  
- "long 저위험 - short 고위험" = 1.050% - 1.291% = **-0.241%/월**
- 음수가 나오는 것은 GARCH나 분류 방법의 문제가 아니라 **시장 구조상 자연스러운 현상**
- ETF 유니버스(SPY, QQQ, XLK, EEM, XLE 등)에서 변동성이 높은 ETF(EEM, XLE, QQQ 등)는 리스크 프리미엄을 받아 장기 평균 수익도 높다
- "저위험 long - 고위험 short" = 리스크 프리미엄을 역방향으로 배팅 → 음수가 당연
- BL 포트폴리오는 long-only이므로 이 팩터를 직접 사용하는 게 아니라, 저위험 종목 over-weight의 방향성만 참고

**`연환산 팩터 수익: GARCH -2.89%/년, Baseline -3.05%/년`**  
- GARCH 팩터가 Baseline보다 연간 0.16%p 손실이 작음
- 절대값이 작을수록 "리스크 프리미엄 역방향 배팅의 비용"이 작다 → GARCH 분류가 상대적으로 낫다는 추가 확인

**`팩터 Sharpe: GARCH -0.197, Baseline -0.213`**  
- 음수 Sharpe는 "이 팩터 자체를 standalone 전략으로 쓰면 안 된다"는 뜻
- 그러나 GARCH(-0.197)가 Baseline(-0.213)보다 덜 나쁨 → GARCH 분류 우위가 여기서도 반영

**`양수 비율: GARCH 51.4%, Baseline 48.6%`**  
- GARCH 팩터는 180개월 중 51.4%(약 92달)에서 양수 수익
- Baseline은 48.6%(약 87달)에서만 양수
- 과반수인 51.4%에서 "저위험이 고위험보다 수익이 높았다" → GARCH 분류 방향성은 절반 이상의 달에서 맞음

### 분석 4: 레짐별 IC 분석 (셀 4)

```
레짐 정의: SPY 12개월 롤링 변동성 (q33=0.107, q67=0.152)

[저변동성] 60개월
  GARCH IC:    0.5631
  Baseline IC: 0.5122
  차이(G-B):   +0.0509  |  GARCH 우위: 44/60 (73.3%)

  GARCH IC:    0.5864
  Baseline IC: 0.5727
  차이(G-B):   +0.0136  |  GARCH 우위: 33/60 (55.0%)

[고변동성] 60개월
  GARCH IC:    0.6130
  Baseline IC: 0.6191
  차이(G-B):   -0.0062  |  GARCH 우위: 27/60 (45.0%)

고변동성 구간 주요 기간:
  {2011: 8, 2012: 8, 2019: 11, 2020: 10, 2021: 3, 2022: 9, 2023: 10, 2024: 1}
```

**이 분석의 목적**:  
시장 레짐(변동성 수준)에 따라 GARCH의 우위가 달라지는지 확인.  
GARCH는 변동성이 높을수록 더 잘 작동할 것으로 예상되지만, 실제는 그 반대.

**레짐 정의**:  
- SPY의 12개월 롤링 변동성(연환산)을 기준으로 3등분
- q33=0.107: 연환산 변동성 10.7% 이하 → 저변동성 (안정된 시장)
- q67=0.152: 연환산 변동성 15.2% 이상 → 고변동성 (스트레스 시장)
- 각 그룹이 정확히 60개월씩 나뉨

**`[저변동성] GARCH IC 0.5631, Baseline IC 0.5122, 차이 +0.0509, GARCH 우위 73.3%`**  
- 시장이 안정적일 때 GARCH가 Baseline보다 0.05 높은 IC → 가장 큰 차이
- 60개월 중 44개월에서 GARCH 승 (73.3%)
- 저변동성 구간에서는 ARCH 클러스터링이 약해지므로, 단순한 vol 이월보다 GARCH 모델이 미래 변동성을 더 잘 포착
- 시장이 조용할 때 GARCH 예측력이 빛남

**`[중간] GARCH IC 0.5864, Baseline IC 0.5727, 차이 +0.0136, GARCH 우위 55.0%`**  
- 중간 변동성 구간에서는 GARCH 우위가 줄어듦 (+0.014)
- 여전히 GARCH가 더 나은 편이지만 차이가 작음
- 과반수인 55%의 달에서 GARCH 승

**`[고변동성] GARCH IC 0.6130, Baseline IC 0.6191, 차이 -0.0062, GARCH 우위 45.0%`**  
- **핵심 역설**: 고변동성 구간에서 두 방법 모두 IC가 가장 높음 (0.61~0.62)
  - 이유: 고변동성 구간에서는 변동성 클러스터링이 극도로 강해져 "지금 변동성 높은 종목 = 다음 달도 높다"는 신호가 매우 선명해짐
  - 즉, 단순 예측 자체가 쉬워지는 구간
- 그러나 GARCH(-0.0062)가 오히려 Baseline보다 낮음
  - 이유: 고변동성 구간에서는 현재 vol_21d 자체가 너무 강한 신호여서 Baseline이 최고의 예측치. GARCH의 추가 파라미터 추정 과정에서 생기는 노이즈가 오히려 미세한 오분류를 유발
- 45.0% 달에서만 GARCH 승 → 이 구간에서는 Baseline이 더 낫다

**`고변동성 구간 주요 기간`**  
- {2011: 8, 2012: 8} → 유럽 재정위기 여진 기간
- {2019: 11} → 미중 무역전쟁 불확실성
- {2020: 10} → 코로나19 급등락
- {2022: 9, 2023: 10} → 연준 금리 인상 사이클
- 이 기간들이 모두 "GARCH보다 Baseline이 나은 고변동성 구간"에 해당

### 분석 6: 종합 결론 셀 (셀 6)

```
① Rank IC 절대값: GARCH 0.5875  Baseline 0.5680  차이 +0.0194
   GARCH 우위 비율: 57.8%

② P 행렬 평균 일치율: 60.6%
```

**`결론 셀의 자동 판정 로직 설명`**:  
코드는 IC 차이, P 일치율, 고변동성 IC 차이 세 가지 조건으로 결론을 자동 생성함.

- `|IC 차이| < 0.01` → "거의 동일" 판정 (0.0194라 이 분기 해당 안 됨)
- `IC 차이 > 0` → "GARCH 우위" 판정 (0.0194로 이 분기 해당)
- `P 일치율 < 0.70` → "상당히 다른 분류" 판정 (60.6%로 이 분기 해당)
- 고변동성 IC 차이는 출력만 되고 자동 판정에는 포함되지 않음 (음수이지만 작은 값)

**`"팩터 스프레드 음수: 분류가 수익으로 이어지지 않음 (재검토 필요)"이라는 출력의 의미`**:  
코드가 자동으로 팩터 스프레드 부호를 체크해서 음수이면 이 경고를 출력.  
그러나 이는 **오해 소지가 있는 자동 메시지**: ETF 유니버스에서 리스크 프리미엄 역방향 배팅으로 인한 음수는 자연스럽다 (분석 3_5 ② 참조). 실제로 BL long-only 포트폴리오에서는 저위험 over-weight가 핵심이지, 이 팩터를 직접 사용하지 않는다.

### GARCH 사용 정당성

GARCH 자체의 성능(IC, Precision, 분리도)은 Baseline 대비 명확히 우위다.  
포트폴리오 Sharpe가 낮아진 이유는 GARCH가 나빠서가 아니라,  
**고변동성 구간에서 Baseline의 예측력이 비정상적으로 강해지기 때문**이다.

이는 변동성 클러스터링(ARCH 효과)의 역설: 클러스터링이 강할수록 현재값이 최고의 예측치.

3. **레짐 전환은 현재 범위 외**: 구현 복잡도가 높아 추후 과제로 분류
