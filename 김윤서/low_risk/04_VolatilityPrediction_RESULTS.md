## 이상치 원인 분석

### ① 2011 QLIKE = 15,224 — 금융위기 학습 잔상

**메커니즘:**

```
훈련 데이터: 2006-01 ~ 2010-12 (60개월)
  └─ 2008-2009 금융위기 포함 (S&P 500 연환산 vol 최대 80%+)

GARCH 추정 결과: 지속성 파라미터 (α + β) → 1에 가깝게 추정
  └─ "위기 기억이 오래 남는" 모델로 학습됨

2011년 실제 시장: 금융위기에서 회복 중, vol 빠르게 정상화

문제 발생:
  예측 종목: SHY (단기 국채 ETF)
    GARCH 예측 vol : ~15% annualized  (위기 데이터 기반)
    실현 vol_21d   : ~1-2% annualized (정상화된 시장)

  QLIKE ratio = ĥ / σ² = (0.15)² / (0.015)² = 100 → QLIKE ≈ 95
  (수백 개 종목 중 일부 채권 ETF에서 ratio 수천 → 월 평균 폭발)
```

**핵심**: 2008-2009 고변동성 체제로 학습된 GARCH가 2011년 저변동성 체제의 채권 ETF(SHY, AGG, TIP 등)를 만나면 분산을 수십~수백 배 과대예측. QLIKE는 이 과대예측에 폭발적으로 반응 (과대예측 → ratio >> 1 → QLIKE ∝ ratio).

**왜 RMSE는 정상(0.1622)인가?**  
RMSE = `(예측 - 실현)` 의 제곱합. 채권 ETF 몇 종목의 오차가 크더라도 500+ 종목 전체 평균에서 희석됨. QLIKE는 ratio의 함수라 분모(σ²)가 작으면 기하급수적으로 커짐.

### ② 2016 QLIKE = 1,266 — vol 급등 후 급락 구간

**메커니즘:**

```
훈련 데이터: 2011-01 ~ 2015-12 (60개월)
  └─ 2015년 8월 중국 발 시장 급락 (VIX 40+ 도달) 포함
  └─ 2015년 내내 elevated vol

GARCH 추정 결과: 2015 스파이크 학습 → 2016 high vol 예측

2016년 실제 시장:
  - 1분기: 추가 하락 (유가 폭락, 중국 우려)
  - 상반기 후반: 급격히 안정화 (VIX 10대 복귀)
  - 6월: Brexit 단기 스파이크 → 빠른 회복
  - 11월: 트럼프 당선 스파이크 → 빠른 회복

결과:
  채권 ETF: GARCH는 2015 스파이크 기반으로 높은 vol 예측
  실제: 2016 중반 이후 채권 vol = 매우 낮음
  → 다시 ratio 폭발
```

**패턴**: 2011과 동일 — vol 급등 구간을 학습한 GARCH가 이후 빠른 정상화를 따라가지 못함. **GARCH의 구조적 약점: 레짐 전환(high→low vol) 감지 지연**.

### ③ 2020 RMSE = 0.2665 (최고) — COVID-19 언더예측

**메커니즘:**

```
훈련 데이터: 2015-01 ~ 2019-12 (60개월)
  └─ 저변동성 시대 ("공포 없는 상승")

GARCH 예측: 낮은 vol 예측 (정상 시장 기준)

2020년 3월 실제: COVID-19 → VIX 82 (2008년 이후 최고)
  실현 vol_21d: 예측치의 수배 → |예측 - 실현| 폭발 → RMSE 급등

QLIKE 관점:
  ratio = ĥ / σ² < 1  (예측 < 실현, 과소예측)
  QLIKE 수식의 비대칭성: 과소예측보다 과대예측에 더 강한 페널티
  → QLIKE 2020 = 0.585 (2011·2016 대비 매우 낮음)
```

**핵심**: RMSE는 예측-실현의 절대 오차를 잡고, QLIKE는 **과대예측**에 훨씬 민감. 2020은 반대 방향(과소예측)이라 RMSE는 폭발하지만 QLIKE는 크지 않음.

## 핵심 발견 요약

### 발견 1: 순위 예측력은 탁월하고 안정적

IC 0.587, ICIR 7.28, 양수 비율 100% — 이 세 수치가 동시에 나오는 것은 GARCH가 종목 간 **상대적 위험도 순서**를 매우 일관되게 맞춘다는 의미. P 행렬 구성(하위 30% vs 상위 30% 분류)의 핵심 목적에 부합.

### 발견 2: GARCH의 실질 역할은 "정확한 수치 예측"이 아님

RMSE 0.1347 = 연환산 vol 기준 13%p 오차. 절댓값으로는 크지만, 이 프로젝트에서 GARCH 예측값은 **P 행렬의 저/고위험 그룹 분류 기준**으로만 쓰임. 분류 기준으로서의 성능은 IC가 측정 — 여기서 0.587로 충분.

### 발견 3: QLIKE가 드러내는 GARCH의 구조적 약점

QLIKE 평균 1099 vs 중앙값 0.38 → **두 개의 다른 분포**가 섞임:
- **정상 구간** (156개월): 중앙값 0.38 수준. GARCH 분산 예측 양호.
- **레짐 전환기** (2011 1년, 2016 일부): QLIKE 폭발. GARCH가 이전 고vol 체제를 "기억"하여 저vol 종목(채권 ETF)을 수십~수백 배 과대예측.

이 취약점은 **08번 Regime3 전략의 이론적 근거**: 고변동성 레짐에서 Q=0 (BL 뷰 비활성화)으로 하는 것이 GARCH 과대예측을 방어하는 구조.

### 발견 4: 2020은 반대 방향 실패

RMSE 0.267 (최고) + QLIKE 0.585 (정상 수준) = GARCH가 COVID-19 vol 급등을 **과소예측**. QLIKE는 과대예측에 민감하고 과소예측에는 덜 민감한 비대칭 손실함수라 QLIKE는 크지 않음. 이는 QLIKE 수식의 설계 의도와 일치.

### GARCH는 이 프로젝트의 목적에 적합

```

GARCH 성능: IC 0.587, ICIR 7.28, 양수 100%

```

### 한계 (논문 제한점으로 명시)

1. **레짐 전환 취약**: 고vol → 저vol 전환 시 GARCH 과대예측 (2011, 2016)
2. **급격한 vol 급등 미예측**: COVID-19 같은 블랙스완 (2020)
3. **채권 ETF QLIKE 왜곡**: 초저변동성 자산에서 과대예측 비율이 극단적

### 08번 Regime3 전략과의 연결

레짐 판별 → 고변동성 레짐에서 Q=0 설정은 GARCH의 레짐 전환 취약성을 **포트폴리오 레벨에서 방어**하는 구조. GARCH 예측이 불안정한 구간(고vol 레짐)에서는 BL 뷰를 비활성화하고 CAPM 균형으로 후퇴.
