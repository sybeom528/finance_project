## 1. 결론 요약

**모멘텀을 BL의 Q(기대수익 뷰)로 사용할 수 없다.**

- IC(Information Coefficient)가 모든 윈도우에서 통계적으로 유의미하지 않음 (≈ 0)
- IC ≈ 0 → Q는 신호가 아닌 노이즈 → BL이 Prior를 개선하지 못하고 왜곡
- 파라미터 조정(6-1 룩백, threshold=0.0)으로도 성능이 개선되지 않음 → 구조적 문제 확인

## 2. IC 분석 결과

### 2-1. 모든 윈도우 IC 값 (Step5_Stocks_Universe_BL.ipynb)

| 모멘텀 윈도우 | IC (Spearman) | 유의성 |
|-------------|--------------|--------|
| 21일 | -0.0317 | ✗ |
| 63일 | -0.0209 | ✗ |
| 126일 | -0.0100 | ✗ |
| 189일 | -0.0013 | ✗ |
| 252일 | -0.0118 | ✗ |
| 6-1 (6개월-1개월) | -0.0082 | ✗ |
| 9-1 | +0.0046 | ✗ |
| 12-1 | -0.0061 | ✗ |

**기준**: Grinold & Kahn(1999) — IC ≥ 0.05가 실용 가능한 신호의 최소 기준

### 2-2. 자산군별 IC 패턴

- **주식(Stocks)**: 일관되게 음수 IC → 모멘텀이 오히려 역방향 예측
- **채권(Bonds)**: 소폭 양수이지만 통계적 유의성 없음
- **대안(Alternatives)**: 불안정, 일부 양수 구간 있으나 노이즈 수준

### 2-3. Non-overlapping IC 계산 방식

```python
# 자기상관 방지를 위해 21일(forward_days) 간격으로 샘플링
step = forward_days  # 21
ic_values = []
for t in range(0, len(dates) - forward_days, step):
    signal = momentum_at_t
    realized = forward_return_at_t
    ic = spearmanr(signal, realized).correlation
    ic_values.append(ic)
```

## 3. 왜 IC ≈ 0인가: 구조적 원인

### 3-1. 시간 스케일 불일치

| 전략 | 신호 기간 | 예측 기간 | 일치 여부 |
|------|---------|---------|---------|
| Antonacci 듀얼 모멘텀 | 12개월 룩백 | 12개월 보유 | ✓ 일치 |
| 우리 프로젝트 | 6~12개월 룩백 | 21일 예측 | ✗ 불일치 |

**비유 (고등학생 수준):**  
축구팀의 "최근 1년 승률"로 "내일 경기 결과"를 예측하는 것. 장기 추세 신호를 단기 예측에 쓰면 맞출 확률이 동전 던지기(50%)와 다를 바 없다.

### 3-2. 횡단면 문제의 난이도

- **Antonacci 원본**: 3개 자산 중 어느 방향인지 (방향성 문제)
- **우리 프로젝트**: 61개 자산 중 누가 더 높을지 순위 예측 (횡단면 랭킹 문제)
- 횡단면 랭킹은 방향성보다 훨씬 높은 IC가 필요 — 실제로 달성 불가

### 3-3. BL 수식에서 IC=0의 의미

```
μ_BL = [(τΣ)⁻¹ + PᵀΩ⁻¹P]⁻¹ × [(τΣ)⁻¹π + PᵀΩ⁻¹Q]
```

- IC ≈ 0 → Q가 무작위 노이즈
- Ω⁻¹ 항 (Q 가중치)이 노이즈를 Prior에 주입
- 결과: BL 사용이 Prior만 쓰는 것보다 **나쁠 수 있음**

## 5. 게리 안토나치 듀얼 모멘텀 vs 우리 프로젝트

### 5-1. 안토나치 전략이 40년간 효과적이었던 이유

| 하락장 | 특성 | 12개월 신호 감지 여부 |
|--------|------|------------------|
| 2000~2002 (닷컴버블) | 점진적, 2년간 하락 | ✓ 2000년 초 감지 |
| 2008~2009 (금융위기) | 점진적, 2007년 말부터 서서히 | ✓ 2008년 3월 이미 신호 발생 |
| 2020 (코로나) | 급격, 33일만에 -34% | ✗ 감지 불가 (2월까지 +20%) |
| 2022 (금리 인상기) | 주식+채권 동시 하락 | △ 피난처 없음 문제 |

- **2008 감지 가능한 이유**: 리먼 사태(9월) 이전에 이미 2007년 말부터 주가 하락 시작 → 2008년 3월 기준 12개월 수익이 이미 음수
- **2020 감지 불가한 이유**: 2020년 1월까지 S&P500 사상최고치 → 2월 19일~3월 23일에 33일만에 급락 → 12개월 룩백으로는 포착 불가

### 5-2. 2022년 "피난처 없음" 문제

```
전통 가정: 주식 하락 → 채권(TLT, AGG) 상승 (안전자산 대피)
2022 현실: 주식(-18%) + 채권(-26%, 금리 급등) 동시 하락
→ 안전자산이 없음
```

- **SHY**: 단기국채이지만 금리 인상기에 SHY 자체도 음수 수익 → 절대 모멘텀 필터에 걸려 편출될 수 있음
- **DBC(원자재)**: 러시아-우크라이나 전쟁으로 2022년 +20% → 피난처 역할 가능
  - 단, DBC는 방어 자산이 아닌 경기순환 자산: 2008년 -35%, 2020년 -25%
  - 우연히 2022년에 수혜를 입은 것이지 구조적 안전자산이 아님

## 6. 한국 논문과의 비교 (이창훈 & 배원성, 2024)

### 6-1. 논문 개요

- **제목**: Black-Litterman 모형을 이용한 듀얼 모멘텀 전략의 성과 분석
- **관측 기간**: 2010년 1월 ~ 2023년 12월 (약 14년)
- **대상 자산**: SPY, TLT, GLD (3개 대표 자산) + 14개 ETF 유니버스

### 6-2. 논문의 모멘텀 활용 방식

**역할 1: Q (BL 기대수익 뷰)**
```
Q = average(r_6m, r_9m, r_12m)   ← 모멘텀 수익률 → Q로 직접 사용
Ω = diag(σ²_6m, σ²_9m, σ²_12m)  ← 분산 → 불확실성으로 사용
```
- 적용 대상: SPY, TLT, GLD **3개**만

**역할 2: 편출 필터 (Dual Momentum)**
```
절대 모멘텀: 12개월 수익 > T-bill → 편입 유지
상대 모멘텀: 자산군 내 상위 ETF 선택 (14개 중)
```

### 6-3. 논문과 우리의 핵심 차이

| 구분 | 논문 | 우리 프로젝트 |
|------|------|--------------|
| Q 적용 자산 수 | **3개** (SPY, TLT, GLD) | **61개** |
| 예측 문제 유형 | 방향성 (오른다/내린다) | 횡단면 순위 (61개 중 누가 더 오를까) |
| IC 검증 | 없음 (선행 연구 가정) | 직접 계산 → 모두 ≈ 0 |
| IC 요구 수준 | 낮음 | 높음 |
| 관측 기간 편향 | 2010~2023 = 대세 상승장 | — |

### 6-4. 논문이 좋은 성과를 낸 이유

1. **관측 기간 편향**: 2010~2023은 코로나 반등 포함한 대세 상승장. 논문 자체도 이 한계를 인정
2. **단순한 예측 문제**: 3개 자산 방향성 → IC 요구치 낮음
3. **IC 미검증**: 모멘텀이 효과있다는 선행 연구를 가정으로 채택
4. **매매비용 없음**: 실제 거래비용 미반영

### 6-5. SPY 기반이었다면 우리도 효과있었을까?

- **가능성 있음**: 61개 → 3개로 줄이면 횡단면 문제 → 방향성 문제로 단순화
- **그러나**: IC를 사전 검증하지 않으면 논문과 동일한 한계를 가짐
- **근본적 문제**: 우리의 21일 예측 지평선이 모멘텀 신호(6~12개월)와 시간 스케일 불일치 — 자산 수를 줄여도 이 문제는 해결되지 않음

## 7. 튜터 설명용 논리 체계

### 7-1. IC ≈ 0 → 모멘텀 Q 사용 불가 논리 흐름

```
[관찰] 모멘텀 IC ≈ 0 (모든 윈도우, 비중첩 샘플)
    ↓
[의미] 모멘텀 신호와 21일 후 수익률 간 상관관계 없음
    ↓
[BL 영향] Q = 노이즈. Ω⁻¹ × Q가 Prior에 노이즈 주입
    ↓
[실증] 6-1 / threshold=0.0 파라미터 변경 후에도 Sharpe 악화
    ↓
[결론] 구조적 문제. 시간 스케일 불일치 + 횡단면 예측의 어려움
    ↓
[대안] 모멘텀 대신 ML 예측 (Step3)을 Q로 사용 → 파이프라인 유지
```

### 7-2. 핵심 메시지

> "Antonacci의 듀얼 모멘텀은 12개월 신호로 12개월을 예측하고 3개 자산의 방향성만 맞추면 된다. 우리는 6~12개월 신호로 21일을 예측하고 61개 자산의 순위를 맞춰야 한다. 이 두 문제의 난이도 차이가 IC의 차이로 나타난다."
