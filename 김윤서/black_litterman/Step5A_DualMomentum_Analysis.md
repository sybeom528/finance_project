## 2. 듀얼 모멘텀 개념 (Antonacci, 2014)

```
Step 1. 절대 모멘텀 (Absolute Momentum) — 하락장 방어
   각 자산의 12-1 수익률 > T-bill 수익률?
   NO  → 편출 (비중 0, 현금 보유)
   YES → 다음 단계

Step 2. 상대 모멘텀 (Relative Momentum) — BL Q 생성
   편출 안 된 자산끼리 횡단면 순위 기반 BL Q 생성
   → MVO로 최종 비중 결정
```

### 12-1 모멘텀 적용 (단기반전 노이즈 제거)

| 항목 | 공식 | 이유 |
|------|------|------|
| 절대 모멘텀 필터 | `P[t-22] / P[t-253] - 1 > T-bill` | 최근 1개월 차익실현 노이즈 제거 |
| Q 생성 (상대 모멘텀) | `(log P[t-21] - log P[t-w]) / (w-21) × 252` | 동일 이유, Fama-French 표준 |

## 5. Sharpe Ratio가 낮아진 이유 (핵심)

### 5-1. 2016-2025 강세장에서 편출의 역효과

절대 모멘텀 필터는 **후행 지표**다. 12-1 수익률이 T-bill 아래로 떨어지려면 이미 오래 하락한 상태여야 한다.

```
강세장에서의 시나리오:
  단기 조정 발생 → 12개월 모멘텀은 여전히 양수 → 보유 유지 ✓
  → 대부분의 경우 편출 안 됨

BUT 강세장 전체에서:
  평균 31% 편출 = 31%를 현금으로 보유
  → 수익률 하락 (현금은 0% 수익)
  → 변동성은 크게 줄지 않음 (남은 69%는 동일한 변동성)
  → Sharpe = 수익 / 변동성 → 분자(수익)만 크게 감소
```

### 5-2. 편출 후 재진입 지연 (모멘텀 크래시)

```
2020년 COVID 시나리오:
  3월 급락 → 12-1 모멘텀 음수 → 편출 (BL 이후 리밸런싱에서)
  4월~12월 급반등 → 12-1 모멘텀 여전히 음수 (12개월 평균이 회복 안 됨)
  → 급락 시 편출 성공, 반등 시 재진입 지연 → 순이익 거의 0
```

### 5-3. 수치로 확인

| 지표 | Step5 | Step5A | 해석 |
|------|-------|--------|------|
| 연수익 (Neutral) | ~18% | 13.46% | 강세장 편출로 -4.5%p |
| MDD (Neutral) | ~-33% | -28.93% | MDD는 소폭 개선 |
| Sharpe | ~0.85 | 0.665 | 수익 감소 > 위험 감소 |
| EqualWeight Sharpe | - | **0.892** | 편출 자산 제외 등가중이 가장 좋음 |

### 5-4. EqualWeight가 Sharpe 0.892로 가장 높은 이유

EqualWeight는 절대 모멘텀으로 편출된 자산을 제외한 **잔존 자산에 등가중** 투자한다. BL의 MVO를 거치지 않아 모멘텀 Q의 오차가 축적되지 않는다. 이것이 이 기간에는 MVO 최적화보다 나은 성과를 낸다.

> **결론**: 절대 모멘텀 필터 자체(편출 방어)는 MDD를 소폭 개선했지만,  
> 2016-2025 강세장에서 31% 평균 현금 보유는 수익률을 더 많이 깎아 Sharpe가 낮아졌다.

## 7. 향후 개선 방향

### 7-1. BL에서 튜터 권고 방향

> "원본 논문 방식 그대로 한 결과와 Z-score 방식을 비교해봐야 개선점이 명확해진다."

| 버전 | Omega 방식 | 현재 상태 |
|------|-----------|---------|
| 원본 논문 | Ω = τ·(P·Σ·Pᵀ) / c | 미구현 (비교 baseline 필요) |
| Step5/5A | Idzorek Z-score | 구현됨 |

### 7-2. 강세장 편출 문제 해결책 후보

```
1. 편출 임계값 완화: T-bill → T-bill - 2%
   (너무 쉽게 편출되지 않도록)

2. 편출 자산을 현금 대신 채권(SHY/AGG)으로 대체
   (0% 수익 대신 채권 수익률 확보)

3. 편출 유예기간 추가:
   2회 연속 음수일 때만 편출 → 일시적 반전에 반응 안 함
```

### 7-3. IC 개선 방향

```
현재 Step5/5A (절대 모멘텀 Q):
  IC ≈ -0.01 ~ -0.03 → 신호 거의 없음

Step 3 ML Q 통합 후:
  XGBoost/TabPFN Q → IC 개선 기대
  → c_max 상향 (0.90 → 0.95)으로 Q 영향력 강화
```
