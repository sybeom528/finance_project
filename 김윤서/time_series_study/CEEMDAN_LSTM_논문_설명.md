# 논문 설명: CEEMDAN + LSTM 주가 예측 모델
> Cao et al. (2019), *Physica A 519*, 127–139

---

## 0. 한 줄 요약

> **"주가 데이터를 잘게 분해해서 각 조각을 AI로 예측한 뒤, 다시 합치면 더 정확하다"**

---

## 1. 왜 이 연구를 했나?

주가는 매일 오르락내리락합니다. 이걸 정확히 예측하면 돈을 벌 수 있겠죠?  
그런데 주가에는 두 가지 큰 어려움이 있어요.

| 어려움 | 설명 |
|--------|------|
| **비선형(Non-linear)** | 단순한 공식으로 설명이 안 됨. 복잡하게 꼬여있음 |
| **비정상(Non-stationary)** | 평균이나 분산이 시간에 따라 계속 변함 |

이전까지도 AI(인공지능)로 주가를 예측하려는 시도는 많았어요.  
예: BP 신경망, SVM, RNN, LSTM 등  
하지만 이 논문은 **"신호 분해"** 라는 기법을 LSTM과 결합해서 성능을 더 높였습니다.

---

## 2. 핵심 아이디어: 분해 → 예측 → 합치기

### 비유로 이해하기

> 피아노 소리를 녹음했다고 생각해 봐요.  
> 거기엔 저음(베이스), 중음(멜로디), 고음(반주)이 섞여 있어요.  
> 이걸 그냥 한꺼번에 분석하면 어렵지만,  
> **주파수별로 분리해서** 각각 분석하면 훨씬 쉽겠죠?  
>
> 주가도 마찬가지입니다!  
> 단기 노이즈(하루하루 등락), 중기 흐름, 장기 트렌드가 섞여 있어요.  
> 이걸 **분해 → 각각 예측 → 다시 합치기** 하는 게 이 논문의 핵심입니다.

---

## 3. Step 1: 신호 분해 — EMD와 CEEMDAN

### EMD (경험적 모드 분해)

EMD는 복잡한 시계열 데이터를 **여러 개의 IMF(고유 모드 함수)** 로 쪼개는 방법입니다.

```
원본 주가 데이터
        ↓
IMF1 (가장 빠른 진동, 노이즈에 가까움)
IMF2 (두 번째로 빠른 진동)
IMF3
...
IMFn
Residue (잔차 = 장기 트렌드)
```

수식으로 쓰면:

```
원본 데이터 S(t) = IMF1 + IMF2 + ... + IMFn + Residue
```

**문제점:** EMD는 "모드 혼합(Mode Mixing)" 현상이 발생해요.  
→ 비슷한 진동수가 서로 다른 IMF에 섞이거나, 하나의 IMF에 완전히 다른 진동수가 같이 들어가는 문제

### EEMD → CEEMDAN으로 개선

| 방법 | 설명 | 문제 |
|------|------|------|
| EMD | 기본 분해 | 모드 혼합 발생 |
| EEMD | 화이트 노이즈를 여러 번 추가해서 평균냄 | 재조합 시 잔류 노이즈 남음 |
| **CEEMDAN** | 노이즈를 단계별로 적응적으로 추가 | **거의 완벽한 재조합, 계산도 빠름** ✅ |

> CEEMDAN = **C**omplete **E**nsemble **EMD** with **A**daptive **N**oise  
> "완전한 앙상블 경험적 모드 분해 + 적응형 노이즈"

CEEMDAN의 핵심 과정 (간단히):
1. 원본에 약간의 랜덤 노이즈를 더해서 여러 번 EMD 분해
2. 결과를 평균 내서 첫 번째 IMF 추출
3. 잔차를 계산하고, 다시 노이즈를 더해서 두 번째 IMF 추출
4. 이 과정을 반복 → 모드 혼합 없이 깨끗하게 분해됨

## 4. Step 2: LSTM으로 각 IMF 예측

분해된 각 IMF마다 **LSTM 모델**을 하나씩 만들어서 예측합니다.

### LSTM이란?

LSTM = **Long Short-Term Memory** (장단기 기억 신경망)

일반 RNN(순환 신경망)의 단점(오래된 정보를 잊어버리는 문제)을 해결한 모델입니다.

**LSTM의 핵심: 3개의 "게이트(문)"**

```
입력 게이트 (Input Gate)   → 지금 새로운 정보를 얼마나 받아들일까?
망각 게이트 (Forget Gate)  → 과거 정보를 얼마나 지울까?
출력 게이트 (Output Gate)  → 지금 뭘 출력할까?
```

> 마치 사람이 공부할 때:  
> - 새로운 내용을 배울지 (입력 게이트)  
> - 오래된 정보를 잊을지 (망각 게이트)  
> - 지금 이 상황에서 뭘 쓸지 (출력 게이트)  
> 를 결정하는 것과 비슷해요!

수식으로 보면:

```
forget gate:  ft = σ(Wf · [h_이전, x_지금] + bf)
input gate:   it = σ(Wi · [h_이전, x_지금] + bi)
cell update:  ct = ft * c_이전 + it * c̃t
output gate:  ot = σ(Wo · [h_이전, x_지금] + bo)
output:       ht = ot * tanh(ct)
```

σ (시그모이드)는 0~1 사이 값을 내보내는 함수 → "얼마나" 통과시킬지 조절

### 이 논문에서 쓴 LSTM 구조

```
Layer 1: LSTM (128차원 출력)
Layer 2: LSTM (64차원 출력)
Layer 3: FC (Fully Connected, 16개 뉴런)
Layer 4: FC (출력 1개 → 예측값)
```

- 활성화 함수: ReLU
- 손실 함수: MSE (평균 제곱 오차)
- 최적화: Adam

### 과적합 방지: Dropout

훈련할 때 일부 뉴런을 **랜덤하게 꺼버립니다** → 특정 패턴에만 과도하게 의존하지 않도록 방지

---

## 5. Step 3: 예측 결과 합치기

각 IMF에 대해 예측한 값들을 다시 더하면 최종 주가 예측값이 됩니다.

```
최종 예측값 = 예측된 IMF1 + 예측된 IMF2 + ... + 예측된 IMFm + 예측된 Residue
```

### 전체 흐름 요약

```
원본 주가
    ↓ [CEEMDAN 분해]
IMF1, IMF2, ..., IMFm, Residue
    ↓ [각각 LSTM 학습 및 예측]
예측 IMF1, 예측 IMF2, ..., 예측 Residue
    ↓ [합산]
최종 예측 주가
```

---

## 6. 실험: 어떻게 검증했나?

### 사용 데이터

4개의 글로벌 주가 지수 (2007년 12월 ~ 2017년 12월, 약 10년치 일별 종가)

| 지수 | 설명 | 데이터 수 |
|------|------|-----------|
| S&P500 | 미국 대표 500개 기업 | 2,518개 |
| HSI | 홍콩 항셍 지수 | 2,465개 |
| DAX | 독일 주가 지수 | 2,535개 |
| SSE | 중국 상하이 지수 | 2,433개 |

- 앞 90%: 훈련 데이터
- 뒤 10%: 테스트 데이터

### 성능 평가 지표 3가지

| 지표 | 수식 | 의미 |
|------|------|------|
| **MAE** | 평균(|실제 - 예측|) | 평균적으로 얼마나 틀렸는지 |
| **RMSE** | √평균((실제 - 예측)²) | 큰 오차에 더 민감한 지표 |
| **MAPE** | 평균(|실제 - 예측| / 실제) × 100 | 퍼센트로 얼마나 틀렸는지 (가장 중요하게 봄) |

→ 세 지표 모두 **작을수록 좋음**

## 7. 결과: 얼마나 좋았나?

### EMD-LSTM vs CEEMDAN-LSTM 비교

| 지수 | 모델 | MAPE |
|------|------|------|
| S&P500 | EMD-LSTM | 0.2098% |
| S&P500 | **CEEMDAN-LSTM** | **0.1617%** ✅ |
| HSI | EMD-LSTM | 0.3418% |
| HSI | **CEEMDAN-LSTM** | **0.2768%** ✅ |

→ CEEMDAN이 EMD보다 항상 더 좋음 (모드 혼합이 없어서)

### 다른 모델들과 비교 (S&P500 기준)

| 모델 | MAPE |
|------|------|
| LSTM만 사용 | 0.6115% |
| SVM | 0.3993% |
| CEEMDAN-SVM | 0.2213% |
| CEEMDAN-MLP | 0.2051% |
| **CEEMDAN-LSTM** | **0.1617%** 🏆 |

→ CEEMDAN-LSTM이 모든 모델 중 가장 정확!

### 선형 회귀 분석 (R² 값)

예측값과 실제값이 얼마나 일치하는지 R²(결정계수)로 확인:

| 지수 | R² |
|------|-----|
| S&P500 | 0.9985 |
| HSI | 0.9982 |
| DAX | 0.9960 |
| SSE | 0.9932 |

→ R² = 1에 가까울수록 완벽한 예측. 모두 **0.99 이상** → 매우 높은 정확도!

## 8. 한계점과 향후 연구

논문 저자들도 인정한 한계:

1. **입력 데이터가 종가 하나뿐** → 거래량, 고가, 저가 등도 활용하면 더 좋을 것
2. **다양한 시간 단위** 적용 가능 (분봉, 주봉 등)
3. 주가 외에도 **날씨, 교통량 예측** 등 다른 시계열 데이터에도 적용 가능

## 9. 이 프로젝트와의 연관성

우리 프로젝트(Black-Litterman 기반 ETF 펀드 구축)에서는:

- **Step 3 ML 모델링**: XGBoost, TabPFN을 사용해 자산 수익률을 예측함
- 이 논문의 CEEMDAN-LSTM처럼 **분해 + 예측** 방식을 결합하면  
  Q(기대수익률 뷰)와 Ω(불확실성)를 더 정밀하게 계산할 수 있을 것
- 단, 우리는 **30개 자산 × walk-forward** 구조이므로 계산 비용 고려 필요
